# p6-情報アップロード - 基础信息

## 页面概述

| 属性 | 值 |
|------|-----|
| 页面ID | p6 |
| 页面名称 | 情報アップロード |
| 页面类型 | 上传页面 |
| 所属模块 | 共通 |
| 设备类型 | PC / SP |

## 功能概要

- パートナーが情報をアップロードできる
- CSVファイルなどの情報更新を行う

## 核心功能模块

### 1. アップロード

| 字段名 | 字段类型 | 说明 |
|--------|----------|------|
| ファイル選択 | 文件上传 | CSVファイルを選択 |
| アップロード | 按钮 | ファイルをアップロード |

### 2. アップロード履歴

| 字段名 | 说明 |
|--------|------|
| アップロード日 | アップロードした日時 |
| ファイル名 | アップロードしたファイル名 |
| ステータス | アップロードの状態 |

## 页面交互

### 导航路径

- 来自 `p0-共通ナビゲーション` 的相关菜单

### 操作流程

1. 点击「ファイル選択」
2. 选择CSVファイル
3. 点击「アップロード」
4. 显示アップロード結果
5. 查看アップロード履歴
