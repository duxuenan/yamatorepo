# p6-情報アップロード - 统计分析

## アップロード统计

### 1. 上传统计

| 统计维度 | 计算方法 | 业务含义 |
|----------|----------|----------|
| アップロード件数 | COUNT(*) | 上传次数 |
| 成功率 | SUCCESS / TOTAL | 上传成功率 |
| 平均処理時間 | AVG(duration) | 处理效率 |

### 2. ファイル类型分布

| 类型 | 说明 | 占比 |
|------|------|------|
| CSV | カンマ区切り | - |
| Excel | .xlsx | - |

### 3. エラー分布

| エラー类型 | 说明 | 対処 |
|------------|------|------|
| フォーマットエラー | ファイル形式不正确 | CSV格式确认 |
| サイズエラー | ファイルサイズ超過 | 压缩文件 |
| データエラー | データ内容错误 | 数据验证 |

### 示例数据

```json
{
  "upload_history": [
    {

    }
  ]
}
```
