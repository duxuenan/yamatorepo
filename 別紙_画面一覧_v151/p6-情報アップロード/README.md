# p6-情報アップロード - README

## 页面概述

**页面ID**: p6
**页面名称**: 情報アップロード
**页面类型**: 上传页面
**所属模块**: 共通
**设备类型**: PC / SP

## 功能简介

- パートナーが情報をアップロードできる
- CSVファイルなどの情報更新を行う

## 核心功能

- ファイルアップロード (CSV/Excel)
- アップロード履歴表示
- アップロード結果确认

## 相关页面

- **上级页面**: p0-共通ナビゲーション

## 操作流程

1. 点击「ファイル選択」
2. 选择CSVファイル
3. 点击「アップロード」
4. 显示アップロード結果
5. 查看アップロード履歴

## ファイル规格

| 项目 | 规格 |
|------|------|
| ファイル类型 | CSV, Excel (.xlsx) |
| 最大サイズ | 10MB |
| 文字コード | UTF-8 |
