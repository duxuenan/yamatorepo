# b5-実績詳細 - 详细分析

## 1. 页面结构分析

### 1.1 页面构成
b5-実績詳細 页面由以下区域构成：

| 区域 | 内容 | 说明 |
|------|------|------|
| 輸送パートナー区域 | ⑴輸送パートナー | 対象輸送パートナー情報 |
| 状態区域 | ⑵状態 | 发注対象進捗状況（発注ID/ステータス/version/指示書送付/承諾） |
| 日別傭車費一覧区域 | ⑶日別傭車費一覧 | 费用表格编辑 |
| 月間運行指示書区域 | ⑷月間運行指示書（確定版） | 確定版のみ |
| アクション履歴区域 | ⑸状態・アクション履歴 | 行为履历 |
| メッセージ区域 | ⑹メッセージ | 消息交流 |

### 1.2 Tabs切换
- 「確定前」：可编辑状态
- 「確定後」：只读显示

## 2. 日別傭車費一覧分析

### 2.1 表格字段
| 字段 | 说明 | 交互 |
|------|------|------|
| 線便コード | 线路代码 | 显示 |
| 線便名 | 线路名称 | 显示 |
| 土/日/月/火/水/木/金 | 日期金额 | 点击编辑 |
| 承認済/未承認/差戻し | 状态 | 显示/切换 |
| version | 版本选择 | プルダウン |
| ＋追加 | 添加行 | 链接 |

### 2.1.1 状态说明
| 状态 | 说明 | 操作 |
|------|------|------|
| 承認済 | 已承认 | 点击后可显示[差戻し]按钮 |
| 未承認 | 未承认 | - |
| 差戻し | 退回 | 从承認済状态点击可退回至未承認 |

### 2.2 编辑功能
| 功能 | 操作 | 说明 |
|------|------|------|
| 金額点击 | 显示吹き出し | 编辑金额/事由/変更後 |
| 強調表示 | 背景強調 | 鼠标over显示内容 |
| 行追加 | 「＋追加」 | 添加新行 |
| 保存 | 「変更を保存」 | 保存编辑内容 |
| 送付 | 「送付」 | 发送日別傭車費一覧 |
| 代理承諾 | 「代理承諾」 | 代替承诺 |

## 3. 月間運行指示書（確定版）

### 3.1 特定功能
- 署名・捺印显示（画面表示のみ）
- 確定版本专属显示
- 只读不可编辑

### 3.2 显示内容
| 项目 | 说明 |
|------|------|
| 月間運行指示書 | 確定版本 |
| 署名栏 | 显示用 |
| 捺印栏 | 显示用 |

## 4. サブアカウント振り分け

### 4.1 功能说明
| 功能 | 操作 | 说明 |
|------|------|------|
| プルダウン | 选择支店/営業所 | 单一选择 |
| チェックボックス | 批量选择 | 一括振り分け |
| 状態表示 | 承認済/未承認/差戻し | 状态切换 |

## 5. アクション履歴
