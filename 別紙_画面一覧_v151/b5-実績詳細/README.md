# b5-実績詳細 画面分析文档

## 文档概述

本文档是对 Excel 文件 `別紙_画面一覧_v151.xlsx` 中 `b5-実績詳細` 工作表的详细分析。

## 画面基本信息

| 项目 | 内容 |
|------|------|
| **工作表名称** | b5-実績詳細 |
| **界面编号** | b5 |
| **画面名称** | 実績詳細 |
| **画面类型** | PC |
| **功能概述** | 业绩详情页面，提供日别傭车费一览编辑和发送功能 |

## 功能描述

b5-実績詳細 是业绩详情页面：

1. **輸送パートナー情報** - 対象輸送パートナー情報表示
2. **状態表示** - 発注対象の進捗状況表示
3. **日別傭車費一覧** - 参照/編集/送付功能
4. **月間運行指示書（確定版）** - 確定版のみ表示
5. **アクション履歴** - 本画面での行動登録
6. **メッセージ** - 輸送パートナーとのメッセージやり取り

## 业务状态区分

### 確定前
- 日別傭車費一覧可编辑
- 送付/代理承諾功能

### 確定後
- 月間運行指示書（確定版）显示
- 署名・捺印显示
- 只读不可编辑

## Tabs功能

| Tab | 内容 | 权限 |
|-----|------|------|
| 確定前 | 日別傭車費一覧 | 可编辑 |
| 確定後 | 確定版指示書 | 只读 |
