# b5-実績詳細 - 导航流程说明

## 1. 页面导航概述

### 1.1 页面定位
b5-実績詳細 是系统的**业绩详情页面**，提供业绩详细信息和日别傭车费一览管理功能。

### 1.2 导航类型
| 导航类型 | 说明 |
|----------|------|
| **详情页面** | 展示和编辑详细数据 |
| **功能页面** | 多功能集成（编辑/送付/消息） |

## 2. 入口分析

### 2.1 直接入口
| 入口方式 | 触发条件 | 目标页面 |
|----------|----------|----------|
| 列表跳转 | b4-実績検索点击公司代码 | b5-実績詳細 |
| URL直接访问 | 携带业绩ID | b5-実績詳細 |

### 2.2 入口参数
| 参数 | 类型 | 说明 |
|------|------|------|
| completedOrderId | string | 业绩ID |

## 3. 出口分析

### 3.1 页面跳转出口
| 当前页面 | 触发条件 | 目标画面 | 说明 |
|----------|----------|----------|------|
| b5-実績詳細 | 返回按钮 | b4-実績検索 | 返回列表 |
| b5-実績詳細 | 点击消息链接 | 相关详情 | 跳转关联页面 |

### 3.2 Tabs切换
| Tab | 内容 | 编辑权限 |
|-----|------|----------|
| 確定前 | 日別傭車費一覧 | 可编辑 |
| 確定後 | 確定版指示書 | 只读 |

                  ▼

│         b5-実績詳細                  │
│      （业绩详情页面）                 │

                  │

         │               │
         ▼               ▼
    ┌─────────┐     ┌─────────┐

```
[編集モード]
        │
        ▼

        │
        ▼

    ┌───┴───┐
    │       │
  保存     取消
    │       │
    ▼       ▼
[保存成功] [返回原状态]
```

## 5. 导航状态管理

### 5.1 页面状态
| 状态 | 条件 | 导航行为 |
|------|------|----------|
| **確定前** | 默认Tab | 可编辑状态 |
| **確定後** | 点击Tab | 只读显示 |
| **编辑模式** | 点击編集 | 可编辑状态 |
| **发送中** | 点击送付 | 显示loading |

### 5.2 状态转换
```
[確定前 Tab]
        │
        ▼
[点击編集]
        │
        ▼
[编辑模式]
        │
    ┌───┴───┐
    │       │
  保存     取消
    │       │
    ▼       ▼
[保存成功] [確定前 Tab]
```
