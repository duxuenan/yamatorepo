# b7-被請求詳細 画面分析文档

## 文档概述

本文档是对 Excel 文件 `別紙_画面一覧_v151.xlsx` 中 `b7-被請求詳細` 工作表的详细分析。

| 项目 | 内容 |

## 功能描述

b7-被請求詳細 是账单详情页面：

1. **基本情報显示** - 被請求年月/会社コード/パートナー名/ステータス
2. **プレビュー** - 請求書サムネイル/拡大画像表示
3. **金額確認** - 請求書合計/実績合計/金額差異
4. **確認OK/NG** - 账单确认功能
5. **代理アップロード** - 代理上传功能
6. **メッセージ** - 与输送パートナー的消息交流
