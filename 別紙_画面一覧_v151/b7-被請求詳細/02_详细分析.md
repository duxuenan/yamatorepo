# b7-被請求詳細 - 详细分析

## 1. 页面结构分析

### 1.1 页面构成
b7-被請求詳細 页面由以下区域构成：

| 区域 | 内容 | 说明 |
|------|------|------|
| 基本信息区域 | ⑴⑵⑶⑷ | 被請求年月/会社コード/パートナー名/ステータス |
| プレビュー区域 | ⑸ | 請求書画像预览 |
| 実績確認区域 | ⑹ | 実績詳細跳转 |
| 金額情報区域 | ⑺⑻⑼ | 合計金額/差異 |
| 更新情報区域 | ⑽⑾⑿ | 更新日時/更新者/ファイル名 |
| 操作区域 | ⒀⒁⒂ | アップロード/確認OK/確認NG |
| メッセージ区域 | ⒃ | 消息交流 |

### 1.2 元素详细说明

#### ⑴被請求年月
| 项目 | 内容 |
|------|------|
| **元素类型** | 表示項目 |
| **功能描述** | 対象の被請求年月を表示する |

#### ⑵会社コード
| 项目 | 内容 |
|------|------|
| **元素类型** | 表示項目 |
| **功能描述** | 輸送パートナーの会社コードを表示する |

#### ⑶輸送パートナー名
| 项目 | 内容 |
|------|------|
| **元素类型** | 表示項目 |
| **功能描述** | 輸送パートナーの名称を表示する |

#### ⑷ステータス
| 项目 | 内容 |
|------|------|
| **元素类型** | 表示項目 |
| **功能描述** | 状況に応じたステータスを表示する |

### 1.3 ステータス详细说明
| 状态 | 条件 | 说明 |
|------|------|------|
| 未提出 | 輸送パートナーが請求書をアップロードしていない | 輸送パートナーが本システムを利用しない場合、ベース側が代理アップロードしていない場合も含む |
| 要確認 | 金額不匹配 | 輸送パートナーがアップロードした請求書の合計金額と本システムが保持する実績の合計金額を突合し、金額が一致しない |
| チェック未 | 未确认 | アップロードされた請求書をチェックしていない（請求詳細画面で「確認OK」ボタンが押下されていない） |
| チェック済 | 已确认 | 請求詳細画面で「確認OK」ボタンが押下されたとき |
| ビズインテグラル登録完了 | 已上传 | ビズインテグラル（別システム）に請求書（PDF）を手動アップロードした ※ステータス変更は被請求管理画面より手動で行う |

#### ⑸プレビュー
| 项目 | 内容 |
|------|------|
| **元素类型** | プレビュー画像 |
| **功能描述** | サムネイル/拡大画像表示 |

### 1.4 プレビュー功能详细说明
| 功能 | 操作 | 说明 |
|------|------|------|
| サムネイル表示 | 画面显示 | アップロードされた請求書のサムネイル画像が表示される |
| 拡大画像表示 | 点击「プレビュー」リンク | アップロードされた請求書の拡大画像がポップアップで表示される |
| 画像切换 | 複数アップロード時 | ページングで画像を切り替えることができる（前へ/次へ或はページ番号選択） |
| 关闭弹窗 | 点击「閉じる」ボタン | 拡大表示を閉じることができる |

## 2. 操作功能分析

### 2.1 請求書アップロード（代理登録）
| 功能 | 说明 |
|------|------|
| **操作** | ボタンを押下→ファイル選択 |
| **权限** | 代理上传功能 |
| **场景** | パートナー未上传时使用 |

### 2.2 確認OK（この内容で確定する）
| 功能 | 说明 |
|------|------|
| **操作** | ボタンを押下 |
| **结果** | ステータス→チェック済 |
| **条件** | 請求書没有问题 |

### 2.3 確認NG（再アップロードを許可する）
| 功能 | 说明 |
|------|------|
| **操作** | ボタンを押下 |
| **结果** | ステータス→差戻し |
| **条件** | 請求書有问题 |
| **后续** | パートナー可再上传 |
