# b10-マスタ管理 画面分析文档

## 文档概述

本文档是对 Excel 文件 `別紙_画面一覧_v151.xlsx` 中 `b10-マスタ管理` 工作表的详细分析。

## 画面基本信息

| 项目 | 内容 |
|------|------|
| **工作表名称** | b10-マスタ管理 |
| **界面编号** | b10 |
| **画面名称** | マスタ管理 |
| **画面类型** | PC |
| **功能概述** | 管理入口页面，提供跳转功能 |

## 功能描述

b10-マスタ管理 是系统管理入口页面：

1. **輸送パートナーマスタ管理** - b11-輸送パートナーマスタ検索へ遷移
2. **アカウントマスタ管理** - b14-ベースアカウントマスタ検索へ遷移

### 入口
