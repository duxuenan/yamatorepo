### 1.2 功能描述
b10-マスタ管理 是管理入口页面，提供两个管理功能的跳转按钮。

### 1.3 技术架构
本文档基于 **前后端分离架构**，详细技术规范请参考：

👉 **[\_公共技术架构规范.md](../_公共技术架构规范.md)**

| 层级 | 技术 |
|------|------|
| 前端 | React 18.x + TypeScript + Ant Design |
| 后端 | - | 无后端逻辑 |

### 1.4 核心功能
- 輸送パートナーマスタ管理への跳转
- アカウントマスタ管理への跳转

### 1.5 适用范围
- 前端开发人员（按钮组件实现）

## 4. 前端实现

### 4.1 页面组件
```typescript
// b10管理入口组件
import { Button, Card } from 'antd';
import { UserOutlined, TeamOutlined } from '@ant-design/icons';
import { useNavigate } from 'react-router-dom';

export const B10MasterPage: React.FC = () => {
  const navigate = useNavigate();

  return (
    <div className="b10-master-container">
      <Card title="マスタ管理">
        <div className="button-group">
          <Button
            type="primary"
            size="large"
            icon={<TeamOutlined />}
            onClick={() => navigate('/master/partner')}
          >
            輸送パートナーマスタ管理
          </Button>
          <Button
            type="default"
            size="large"
            icon={<UserOutlined />}
            onClick={() => navigate('/master/account')}
          >
            アカウントマスタ管理
          </Button>
        </div>
      </Card>
    </div>
  );
};
```
