## 用户操作流程图

```
┌─────────────────────────────────────────────────────────────────┐
│                        用户操作流程                              │
└─────────────────────────────────────────────────────────────────┘

[开始]
  │
  ├─► 访问 b14 页面
  │     │
  │     ├─► [搜索账户]
  │     │      │
  │     │      ├─► 输入搜索条件（可输入多个）
  │     │      └─► 点击「検索」
  │     │                │
  │     │                └─► 显示搜索结果
  │     │
  │     ├─► [查看详情]
  │     │      │
  │     │      └─► 点击用户ID链接
  │     │                │
  │     │                └─► 跳转到 b15 页面
  │     │
  │     └─► [新增账户]
  │            │
  │            └─► 点击「＋新規追加」
  │                      │
  │                      └─► 显示用户搜索弹窗
  │                                │
  │                                ├─► 输入用户ID
  │                                ├─► 点击「検索」
  │                                └─► 点击「決定」
  │                                          │
  │                                          └─► 跳转到 b15 页面
  │
  └─► [结束]
```

## 页面状态管理

### URL参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| user_id | string | 预设用户ID |
| user_name | string | 预设用户名 |
| affiliation | string | 预设所属 |
| permission | string | 预设权限 |

### 页面状态

| 状态类型 | 状态值 | 说明 |
|----------|--------|------|
| search | 搜索模式 | 默认状态 |
| result | 结果模式 | 显示搜索结果 |
| empty | 空结果 | 无搜索结果 |
