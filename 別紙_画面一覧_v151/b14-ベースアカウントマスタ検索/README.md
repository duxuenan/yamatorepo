# b14-ベースアカウントマスタ検索 - README

## 页面概述

**页面ID**: b14
**页面名称**: ベースアカウントマスタ検索
**所属模块**: マスタ管理
**设备类型**: PC

## 功能简介

- アカウント情報を検索できる
- 本画面からアカウント情報の詳細に遷移する

## 核心功能

### 1. アカウント検索
- ユーザーID: AzureADに登録されているユーザーIDを入力する
- メールアドレス: AzureADに登録されているメールアドレスを入力する
- ユーザー名: AzureADに登録されているユーザー名を入力する
- 所属: 任意で登録した所属ベース名を入力する
- 権限: ●管理者 / 〇役割者 / 〇配車担当者 / 〇計上担当者

### 2. 検索結果
- ユーザーID: AzureADに登録されているユーザーIDを表示する（クリックでb15画面に遷移）
- ユーザー名: AzureADに登録されているユーザー名を表示する
- 所属: 任意で登録した所属ベース名を表示する
- 権限: 本システムで設定した権限を表示する
- アカウント有効・無効: アカウントの状態が有効か無効かを表示する
- メールアドレス: AzureADに登録されているメールアドレスを表示する

## 操作说明

1. 输入搜索条件（可输入多个条件组合搜索）
2. 点击「検索」按钮执行搜索
3. 在搜索结果列表中点击用户ID查看详情
4. 点击「＋新規追加」新增账户

## 数据来源

- 用户信息: AzureAD
- 所属信息: 本システムで設定
- 権限情報: 本システムで設定
