# b14-ベースアカウントマスタ検索 - 基础信息

## 页面概述

| 属性 | 值 |
|------|-----|
| 页面ID | b14 |
| 页面名称 | ベースアカウントマスタ検索 |
| 页面类型 | 搜索页面 |
| 所属模块 | マスタ管理 |
| 设备类型 | PC |

## 功能概要

- アカウント情報を検索できる
- 本画面からアカウント情報の詳細に遷移する

## 核心功能模块

### 1. アカウント検索

#### 搜索字段

| 字段名 | 字段类型 | 必填 | 说明 |
|--------|----------|------|------|
| ユーザーID | 文本 | 否 | AzureADに登録されているユーザーIDを入力する |
| メールアドレス | 文本 | 否 | AzureADに登録されているメールアドレスを入力する |
| ユーザー名 | 文本 | 否 | AzureADに登録されているユーザー名を入力する |
| 所属 | 文本 | 否 | 任意で登録した所属ベース名を入力する |
| 権限 | radio | 否 | 本システムで設定した権限を選択する（ラジオボタンで選択） |

#### 权限选项

| 选项 | 说明 |
|------|------|
| ● 管理者 | 管理者権限 |
| 〇 役割者 | 役割者権限 |
| 〇 配車担当者 | 配車担当者権限 |
| 〇 計上担当者 | 計上担当者権限 |

### 2. 検索結果

#### 列表字段

| 字段名 | 说明 |
|--------|------|
| ユーザーID | AzureADに登録されているユーザーIDを表示する。クリックするとb15画面に遷移する |
| ユーザー名 | AzureADに登録されているユーザー名を表示する |
| 所属 | 任意で登録した所属ベース名を表示する |
| 権限 | 本システムで設定した権限を表示する |
| アカウント有効・無効 | アカウントの状態が有効か無効かを表示する |
| メールアドレス | AzureADに登録されているメールアドレスを表示する |

### 3. 操作按钮

| 按钮名称 | 功能说明 |
|----------|----------|
| 検索 | 検索条件にマッチしたアカウント情報を一覧で表示する |
| ＋新規追加 | ユーザーID検索のポップアップを表示する |

## 页面交互

### 导航路径

- 来自 `b0-共通ナビゲーション` 的「アカウントマスタ管理」菜单点击

### 页面跳转

- 用户ID链接点击 → `b15-ベースアカウントマスタ詳細`

### 搜索流程

1. 输入搜索条件
2. 点击「検索」按钮
3. 显示符合条件的账户信息列表
