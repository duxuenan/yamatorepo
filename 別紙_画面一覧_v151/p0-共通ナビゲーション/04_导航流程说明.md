# p0-共通ナビゲーション - 导航流程说明

## 页面在系统中的位置

```
輸送パートナーシステム
├── 共通ナビゲーション (p0) - 当前页面
│   ├── 受注管理 → p2-発注検索
│   ├── 実績確認 → p4-実績確認
│   ├── 請求確認 → p5-請求検索
│   └── ログアウト → p1-スタート
```

## 进入当前页面的导航路径

### 路径1：登录后自动进入

```
パートナー用户访问系统
  ↓
完成登录认证
  ↓
自动进入共通导航页面 (p0)
  ↓
显示导航菜单
```

### 路径2：从任意页面返回导航

```
パートナー用户在任意页面
  ↓
点击左侧菜单导航项
  ↓
返回 p0 页面
```

## 从当前页面出去的导航路径

### 跳转到其他页面

| 目标页面 | 触发条件 | 跳转方式 |
|----------|----------|----------|
| p2-発注検索 | 点击「受注管理」 | 直接跳转 |
| p4-実績確認 | 点击「実績確認」 | 直接跳转 |
| p5-請求検索 | 点击「請求確認」 | 直接跳转 |
| p1-スタート | 点击「ログアウト」 | 直接跳转（登出） |

## 面包屑导航

```
ホーム > 共通ナビゲーション
```

## 用户操作流程图

```
┌─────────────────────────────────────────────────────────────────┐
│                        用户操作流程                              │
└─────────────────────────────────────────────────────────────────┘

[开始]
  │
  ├─► 登录系统
  │     │
  │     └─► [进入 p0 页面]
  │                │
  │                ├─► [导航到订单管理]
  │                │      │
  │                │      └─► 点击「受注管理」
  │                │                │
  │                │                └─► 跳转到 p2-発注検索
  │                │
  │                ├─► [导航到実績確認]
  │                │      │
  │                │      └─► 点击「実績確認」
  │                │                │
  │                │                └─► 跳转到 p4-実績確認
  │                │
  │                ├─► [导航到請求確認]
  │                │      │
  │                │      └─► 点击「請求確認」
  │                │                │
  │                │                └─► 跳转到 p5-請求検索
  │                │
  │                └─► [登出系统]
  │                       │
  │                       └─► 点击「ログアウト」
  │                                 │
  │                                 └─► 跳转到 p1-スタート
  │
  └─► [结束]
```

## 页面状态管理

### URL参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| current_page | string | 当前高亮的页面 |

### 页面状态

| 状态类型 | 状态值 | 说明 |
|----------|--------|------|
| active_menu | 受注管理 / 実績確認 / 請求確認 | 当前高亮菜单项 |
| device_type | PC / SP | 设备类型 |
| menu_open | true / false | SP端菜单是否展开 |
