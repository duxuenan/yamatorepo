# p0-共通ナビゲーション - README

## 页面概述

**页面ID**: p0
**页面名称**: 共通ナビゲーション
**页面类型**: 导航页面
**所属模块**: 共通
**设备类型**: PC / SP

## 功能简介

- 主要ページへのリンクを共通メニューとして常時表示してナビゲーションする
- SPの場合、画面右上のハンバーガーメニューを押下するとナビゲーションを表示する

## 核心功能

### 导航菜单
- **受注管理**: → p2-発注検索
- **実績確認**: → p4-実績確認
- **請求確認**: → p5-請求検索
- **ログアウト**: → p1-スタート（ログアウト）

## 设备适配

| 设备 | 导航显示方式 |
|------|-------------|
| PC | 左侧常时显示导航菜单 |
| SP | 右上角汉堡菜单，点击后显示弹窗 |

## 特殊业务规则

- 登录后自动显示导航页面
- 当前页面对应菜单项高亮显示
- 登出后清除会话并跳转到p1页面
