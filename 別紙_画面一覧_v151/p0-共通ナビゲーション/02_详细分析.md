# p0-共通ナビゲーション - 详细分析

## 页面结构层次

```
輸送パートナーシステム
├── 共通ナビゲーション (p0)
│   ├── 受注管理
│   │   └── → p2-発注検索
│   ├── 実績確認
│   │   └── → p4-実績確認
│   ├── 請求確認
│   │   └── → p5-請求検索
│   └── ログアウト
│       └── → p1-スタート
```

## 导航功能说明

### PC端布局

| 组件 | 说明 |
|------|------|
| 左侧菜单栏 | 常时显示所有导航选项 |
| 水平菜单 | 横排列显示菜单项 |
| 高亮当前页 | 当前页面对应菜单高亮 |

### SP端布局

| 组件 | 说明 |
|------|------|
| 汉堡菜单图标 | 屏幕右上角三横线图标 |
| 弹出菜单 | 点击后从右侧滑出或显示弹窗 |
| 全屏覆盖 | 遮罩层覆盖当前页面 |

## 交互流程详细说明

### 1. PC端导航流程

```
页面加载完成
  ↓
显示左侧导航菜单
  ↓
高亮当前页面菜单项
  ↓
用户点击菜单项
  ↓
跳转到对应页面
```

### 2. SP端导航流程

```
页面加载完成
  ↓
显示汉堡菜单图标（右上角）
  ↓
用户点击汉堡菜单
  ↓
显示导航弹窗
  ↓
用户点击菜单项
  ↓
关闭导航弹窗
  ↓
跳转到对应页面
```

### 3. 登出流程

```
用户点击「ログアウト」
  ↓
显示确认对话框（可选）
  ↓
清除登录状态
  ↓
跳转到 p1-スタート 页面
  ↓
显示登录界面
```

## 菜单项详细说明

| 菜单标识 | 菜单名称 | 目标页面 | 业务功能 |
|----------|----------|----------|----------|
| nav_orders | 受注管理 | p2-発注検索 | パートナーの発注を検索・確認 |
| nav_results | 実績確認 | p4-実績確認 | パートナーの確認 |
| nav_billing | 請求確認 | p5-請求検索 | パートナーの請求を検索・確認 |
| nav_logout | ログアウト | p1-スタート | ログアウトして再認証へ |
