# p0-共通ナビゲーション (b0 equivalent for partners) - 基础信息

## 页面概述

| 属性 | 值 |
|------|-----|
| 页面ID | p0 |
| 页面名称 | 共通ナビゲーション |
| 页面类型 | 导航页面 |
| 所属模块 | 共通 |
| 设备类型 | PC / SP |

## 功能概要

- 主要ページへのリンクを共通メニューとして常時表示してナビゲーションする
- SPの場合、画面右上のハンバーガーメニューを押下するとナビゲーションを表示する

## 核心功能模块

### 1. 导航菜单

| 菜单项 | 目标页面 | 说明 |
|--------|----------|------|
| 受注管理 | p2-発注検索 | パートナーの発注検索画面 |
| 実績確認 | p4-実績確認 | パートナーの確認画面 |
| 請求確認 | p5-請求検索 | パートナーの請求検索画面 |
| ログアウト | p1-スタート | ログアウトしてスタート画面に戻る |

## 页面交互

### 导航路径

- 登录后自动显示此导航页面
- SP设备通过点击右上角汉堡菜单显示导航

### 页面跳转

| 触发条件 | 目标页面 |
|----------|----------|
| 点击「受注管理」 | p2-発注検索 |
| 点击「実績確認」 | p4-実績確認 |
| 点击「請求確認」 | p5-請求検索 |
| 点击「ログアウト」 | p1-スタート（ログアウト状態） |

## PC与SP差异

| 功能 | PC | SP |
|------|-----|-----|
| 导航显示 | 常时显示在左侧 | 汉堡菜单点击后显示 |
| 菜单布局 | 水平/垂直列表 | 弹窗式菜单 |
