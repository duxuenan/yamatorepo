# b12-輸送パートナーマスタ詳細（会社）- 基础信息

## 页面概述

| 属性 | 值 |
|------|-----|
| 页面ID | b12 |
| 页面名称 | 輸送パートナーマスタ詳細（会社） |
| 页面类型 | 详情页面 |
| 所属模块 | マスタ管理 |
| 设备类型 | PC |

## 功能概要

- 輸送パートナーの会社情報が確認できる
- 各種アカウントの設定ができる

## 核心功能模块

### 1. 輸送パートナー会社｜基本情報

#### 输入字段

| 字段名 | 字段类型 | 必填 | 说明 |
|--------|----------|------|------|
| 会社名 | 文本 | 是 | 任意の会社名を入力する |
| 会社コード | 文本 | 是 | 任意の会社コードを入力する。カンマ区切りで複数登録可能 |
| 会社電話番号 | 文本 | 是 | 任意の電話番号を入力する（ハイフンなし） |

### 2. アカウント設定

#### 开关选项

| 字段名 | 选项 | 说明 |
|--------|------|------|
| システム利用 | ● あり / 〇 なし | 輸送パートナーが本システムを利用するか、しないかを設定する |

#### 选择项

| 字段名 | 选项 | 说明 |
|--------|------|------|
| 指示書表示文言 | ● 文言1 / 〇 文言2 / 〇 文言3 | 指示書に表示する文言を選択する |

### 3. 利用者一覧

#### 列表字段

| 字段名 | 说明 |
|--------|------|
| ユーザーID | ネコーレWEBに登録されているユーザーIDを表示する |
| アカウント区分 | この利用がメインアカウントなのか、サブアカウントなのかを表示する |
| 支店・営業所名 | 本システムで設定した支店・営業所名を表示する |
| アカウント有効・無効 | このユーザーのアカウントが有効なのか、無効なのかを表示する |
| メールアドレス | ネコーレWEBに登録されているメールアドレスを表示する |

## 页面交互

### 导航路径

- 来自 `b11-輸送パートナーマスタ検索` 页面的用户ID链接点击

### 页面跳转

- 用户ID链接点击 → `b13-輸送パートナー詳細（ユーザー）`

### 操作按钮

| 按钮名称 | 功能说明 |
|----------|----------|
| 保存 | 設定した内容を保存する |
| ＋新規追加 | ユーザーID検索のポップアップを表示する |
| 文言を確認する | 指示書に表示する文言が確認できる（ポップアップ表示） |
