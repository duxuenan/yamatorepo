# b12-輸送パートナーマスタ詳細（会社）- 详细分析

## 页面结构层次

```
配車発注管理システム
├── マスタ管理
│   ├── 輸送パートナー会社｜基本情報
│   │   ├── 会社名
│   │   ├── 会社コード
│   │   └── 会社電話番号
│   ├── アカウント設定
│   │   ├── システム利用
│   │   └── 指示書表示文言
│   └── 利用者一覧
│       ├── ＋新規追加
│       └── 用户列表
```

## 字段详细说明

### 基本信息区域

| 字段标识 | 字段名称 | 数据类型 | 验证规则 | 业务规则 |
|----------|----------|----------|----------|----------|
| company_name | 会社名 | string | 必填，长度限制 | 任意の会社名を入力する |
| company_code | 会社コード | string | 必填，格式验证 | カンマ区切りで複数登録可能 (例: P0001,P0002,P0003) |
| company_phone | 会社電話番号 | string | 必填，数字验证 | 電話番号は(-)ハイフンなし (例: 0312345678) |

### 账户设置区域

| 字段标识 | 字段名称 | 数据类型 | 选项值 | 业务规则 |
|----------|----------|----------|--------|----------|
| system_use | システム利用 | radio | ● あり / 〇 なし | 輸送パートナーが本システムを利用するか、しないかを設定する |
| instruction_text | 指示書表示文言 | radio | ● 文言1 / 〇 文言2 / 〇 文言3 | 指示書に表示する文言を選択する |

### 利用者列表区域

| 字段标识 | 字段名称 | 数据类型 | 来源 | 业务规则 |
|----------|----------|----------|------|----------|
| user_id | ユーザーID | string | ネコーレWEB | ネコーレWEBに登録されているユーザーIDを表示する。リンク押下時はb13画面に遷移 |
| account_type | アカウント区分 | enum | システム | メイン / サブ |
| branch_name | 支店・営業所名 | string | 本システム | 本システムで設定した支店・営業所名を表示する |
| account_status | アカウント有効・無効 | enum | システム | 有効 / 無効 |
| email | メールアドレス | string | ネコーレWEB | ネコーレWEBに登録されているメールアドレスを表示する |

## 交互流程详细说明

### 1. 页面加载流程

```
开始
  ↓
加载基本信息（会社名、会社コード、会社電話番号）
  ↓
加载账户设置（システム利用、指示書表示文言）
  ↓
加载利用者一覧
  ↓
显示完整页面
```

### 2. 保存操作流程

```
用户修改设置
  ↓
点击「保存」按钮
  ↓
验证必填字段
  ↓
保存设置到数据库
  ↓
显示保存成功提示
  ↓
流程结束
```

### 3. 新規追加操作流程

```
点击「＋新規追加」按钮
  ↓
显示用户ID搜索弹窗
  ↓
输入用户ID
  ↓
点击「検索」按钮
  ↓
显示搜索结果
  ↓
点击「決定」按钮
  ↓
跳转到b13-輸送パートナー詳細（ユーザー）页面
```

## 弹窗详细说明

### 指示書表示文言确认弹窗

| 属性 | 值 |
|------|-----|
| 触发方式 | 点击「文言を確認する」链接 |
| 显示内容 | 指示書に表示する文言を確認できる |
| 关闭方式 | 弹窗关闭按钮 |

### ユーザーID搜索弹窗

| 属性 | 值 |
|------|-----|
| 触发方式 | 点击「＋新規追加」按钮 |
| 搜索字段 | ユーザーID |
| 操作按钮 | 検索 / 決定 |
| 搜索结果 | 显示符合条件的用户列表 |
| 后续操作 | 决定后跳转到b13页面 |
