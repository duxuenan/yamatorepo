# b12-輸送パートナーマスタ詳細（会社）

## 页面概述

**页面ID**: b12
**页面名称**: 輸送パートナーマスタ詳細（会社）
**所属模块**: マスタ管理
**设备类型**: PC

## 功能简介

- 輸送パートナーの会社情報が確認できる
- 各種アカウントの設定ができる

## 核心功能

### 1. 基本信息管理
- 会社名: 任意の会社名を入力する
- 会社コード: 任意の会社コードを入力する（カンマ区切りで複数登録可能）
- 会社電話番号: 任意の電話番号を入力する（ハイフンなし）

### 2. 账户设置
- システム利用: ● あり / 〇 なし
- 指示書表示文言: ● 文言1 / 〇 文言2 / 〇 文言3

### 3. 利用者列表
- ユーザーID
- アカウント区分
- 支店・営業所名
- アカウント有効・無効
- メールアドレス
