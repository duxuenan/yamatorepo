## 进入当前页面的导航路径

### 路径1：从搜索页面进入

```
用户访问
  ↓
b11-輸送パートナーマスタ検索 页面
  ↓
点击用户ID链接
  ↓
进入 b12-輸送パートナーマスタ詳細（会社） 页面
```

### 路径2：从列表页面进入

```
用户访问
  ↓
b11-輸送パートナーマスタ検索 页面
  ↓
点击「詳細」按钮
  ↓
进入 b12-輸送パートナーマスタ詳細（会社） 页面
```

## 页面状态管理

### URL参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| partner_company_id | string | 合作伙伴公司ID |
| user_id | string | 当前用户ID（用于权限验证） |

### 页面状态

| 状态类型 | 状态值 | 说明 |
|----------|--------|------|
| view | 浏览模式 | 默认状态，仅查看信息 |
| edit | 编辑模式 | 点击编辑按钮后进入 |
| add | 新增模式 | 新增用户时进入 |
