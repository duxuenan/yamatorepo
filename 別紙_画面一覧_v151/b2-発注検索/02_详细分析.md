# b2-発注検索 - 详细分析

## 1. 页面结构分析

### 1.1 页面构成
b2-発注検索 页面由以下区域构成：

| 区域 | 内容 | 说明 |
|------|------|------|
| お知らせ区域 | information | 系统通知信息展示 |
| 検索条件区域 | 発注検索 | 搜索条件输入 |
| 検索結果区域 | 列表展示 | 订单列表和详情链接 |

### 1.2 元素详细说明

#### ⑴ information
| 项目 | 内容 |
|------|------|
| **元素类型** | 通知区域 |
| **功能描述** | 显示系统通知 |
| **交互功能** | 点击跳转到相关详情页面 |

#### ⑵ 発注検索
| 项目 | 内容 |
|------|------|
| **元素类型** | 搜索区域 |
| **包含条件** | 年月選択、輸送パートナーID、会社コード |
| **交互功能** | 输入条件后点击搜索按钮 |

#### ⑶ 検索結果
| 项目 | 内容 |
|------|------|
| **元素类型** | 列表区域 |
| **显示内容** | 公司代码、公司名、サブアカウント等 |
| **交互功能** | 点击公司代码跳转到b3-発注詳細 |

## 2. 搜索条件分析

### 2.1 条件清单
| 条件名称 | 输入类型 | 说明 |
|----------|----------|------|
| 年月選択 | プルダウン | 选择対象年月 |
| 輸送パートナーID | 入力 | 輸入輸送パートナーID |
| 会社コード | 入力 | 輸入輸送パートナーの会社コード |

### 2.2 筛选功能
| 筛选项目 | 筛选选项 | 说明 |
|----------|----------|------|
| ステータス | 全承諾/未承諾 | 承诺状态筛选 |
| メッセージ | 未読あり | 消息状态筛选 |

## 3. 列表字段分析

### 3.1 显示字段
| 字段名 | 说明 | 交互 |
|--------|------|------|
| 会社コード | 輸送パートナーのコード | クリック→b3-発注詳細 |
| 会社名 | 輸送パートナーの名称 | - |
| サブアカウント | 保有サブアカウント名 | - |
| メッセージ | 未読状態表示 | - |
| ステータス | 承諾状態 | - |
| 最終データ更新日時 | 更新日時 | - |
| version | 最新version | - |
| 一斉メール送信 | 送信日付 | - |
| 承諾状況 | 承諾件数 | - |

### 3.2 显示规则
| 字段 | 显示规则 |
|------|----------|
| メッセージ | 未読あり表示、未読がない場合は「-」 |
| 一斉メール送信 | 未送信は「-」表示 |
