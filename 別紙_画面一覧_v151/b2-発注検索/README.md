# b2-発注検索 画面分析文档

## 文档概述

本文档是对 Excel 文件 `別紙_画面一覧_v151.xlsx` 中 `b2-発注検索` 工作表的详细分析。

## 画面基本信息

| 项目 | 内容 |
|------|------|
| **工作表名称** | b2-発注検索 |
| **界面编号** | b2 |
| **画面名称** | 発注検索 |
| **画面类型** | PC |
| **功能概述** | 登录成功后显示的TOP页面，提供订单搜索和列表展示 |

## 功能描述

b2-発注検索 是系统的首页：

1. **TOP页面** - AzureADでのログインに成功すると、自動的に本システムへ遷移したときに表示される画面
2. **検索功能** - 発注情報の検索＆一覧表示
3. **详情跳转** - 発注情報の詳細画面に遷移できる

## 核心功能

| 序号 | 功能名称 | 功能说明 |
|------|----------|----------|
| 1 | TOP页面 | 登录成功后自动显示 |
| 2 | 信息通知 | お知らせ显示 |
| 3 | 订单搜索 | 年月選択/ID/会社コード条件 |
| 4 | 订单列表 | 9字段详细信息展示 |
| 5 | 详情跳转 | 点击公司コード→b3-発注詳細 |

## 页面状态

### 計画表アップロード前
- 显示"｛対象年}年{対象月}月のデータはありません"

### 計画表アップロード後
- 显示订单搜索区域
- 显示订单列表
- 可进行筛选操作

## 搜索条件

| 条件 | 类型 | 说明 |
|------|------|------|
| 年月選択 | プルダウン | 选择対象年月 |
| 輸送パートナーID | 入力 | 輸入ID |
| 会社コード | 入力 | 会社コード |

## 列表字段

| 字段 | 说明 | 交互 |
|------|------|------|
| 会社コード | 輸送パートナーのコード | クリック→b3-発注詳細 |
| 会社名 | 輸送パートナーの名称 | - |
| サブアカウント | 保有サブアカウント名 | - |
| メッセージ | 未読状態 | - |
| ステータス | 承諾状態 | - |
| 最終データ更新日時 | 更新日時 | - |
| version | 最新version | - |
| 一斉メール送信 | 送信日付 | - |
| 承諾状況 | 承諾件数 | - |
