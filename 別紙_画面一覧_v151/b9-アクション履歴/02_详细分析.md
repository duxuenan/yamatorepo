# b9-アクション履歴 - 详细分析

## 1. 页面结构分析

### 1.1 页面构成
b9-アクション履歴 页面由以下区域构成：

| 区域 | 内容 | 说明 |
|------|------|------|
| 検索条件区域 | ⑴アクション検索 | 搜索条件输入 |
| 検索結果区域 | ⑵検索結果 | 历史记录列表 |

### 1.2 元素详细说明

#### ⑴アクション検索
| 项目 | 内容 |
|------|------|
| **元素类型** | 搜索区域 |
| **包含条件** | 年月選択/会社コード/パートナー名 |

#### ⑵検索結果
| 项目 | 内容 |
|------|------|
| **元素类型** | 列表区域 |
| **显示内容** | 操作历史相关字段 |

## 2. 搜索条件分析

### 2.1 条件清单
| 条件名称 | 输入类型 | 说明 |
|----------|----------|------|
| 年月選択 | プルダウン | 选择対象年月 |
| 会社コード | 入力 | 輸送パートナーの会社コード |
| パートナー名 | 入力 | 輸送パートナーの名称 |

## 3. 列表字段分析

### 3.1 显示字段
| 字段名 | 说明 |
|--------|------|
| 会社コード | 輸送パートナーの会社コード |
| パートナー名 | 輸送パートナーの名称 |
| 日時 | アクションを行った日時 |
| 更新者 | アクションを行ったユーザー名 |
| 項目 | アクション内容 |
