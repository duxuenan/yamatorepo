# b9-アクション履歴 画面分析文档

## 文档概述

本文档是对 Excel 文件 `別紙_画面一覧_v151.xlsx` 中 `b9-アクション履歴` 工作表的详细分析。

## 画面基本信息

| 项目 | 内容 |
|------|------|
| **工作表名称** | b9-アクション履歴 |
| **界面编号** | b9 |
| **画面名称** | アクション履歴 |
| **画面类型** | PC |
| **功能概述** | 系统操作历史查询页面 |

## 功能描述

b9-アクション履歴 是操作历史查询页面：

1. **アクション検索** - アクション履歴を検索
2. **検索結果表示** - 検索結果を一覧で表示

## 核心功能

| 序号 | 功能名称 | 功能说明 |
|------|----------|----------|
| 1 | アクション検索 | 条件搜索（年月/コード/名前） |
| 2 | 検索結果 | 5字段列表展示 |

## 搜索条件

| 条件 | 类型 | 说明 |
|------|------|------|
| 年月選択 | プルダウン | 选择対象年月 |
| 会社コード | 入力 | 会社コード |
| パートナー名 | 入力 | パートナー名 |

## 列表字段

| 字段 | 说明 |
|------|------|
| 会社コード | 輸送パートナーの会社コード |
| パートナー名 | 輸送パートナーの名称 |
| 日時 | アクションを行った日時 |
| 更新者 | アクションを行ったユーザー名 |
| 項目 | アクション内容 |
