# p4-実績確認 - 统计分析

## 実績统计

### 1. 確認统计

| 统计维度 | 计算方法 | 业务含义 |
|----------|----------|----------|
| 確認率 | COUNT(確認済) / COUNT(*) | 确认完成率 |
| 未確認数 | COUNT(WHERE status != '確認済') | 待确认数量 |
| 平均確認時間 | AVG(確認-完了) | 确认处理时效 |

### 2. 状态分布

| 状态 | 说明 | 占比 |
|------|------|------|
| 未確認 | パートナー未確認 | - |
| 確認済 | パートナー確認済 | - |

### 示例数据

```json
{
  "results": [
    {
      "result_id": "RES-2024-001",
      "status": "確認済",
      "confirmed_at": "2024-01-15 12:00:00"
    }
  ]
}
```
