# p4-実績確認 - 基础信息

## 页面概述

| 属性 | 值 |
|------|-----|
| 页面ID | p4 |
| 页面名称 | 実績確認 |
| 页面类型 | 详情页面 |
| 所属模块 | 実績管理 |
| 设备类型 | PC / SP |

## 功能概要

- パートナーが確認すべき実績情報を表示する
- ンベースからの依頼に対する実績を確認する

## 核心功能模块

### 1. 実績情報

| 字段名 | 说明 |
|--------|------|
| 実績ID | 実績の識別子 |
| 受注ID | 関連する発注ID |
| 年月 | 実績の年月 |
| 依頼ベース | 依頼元ベース名 |

### 2. 実績詳細

| 字段名 | 说明 |
|--------|------|
| 実績内容 | 具体的な実績内容 |
| 完了日 | 実績完了日 |
| ステータス | 実績のステータス |

### 3. パートナー確認

| 按钮名称 | 功能说明 |
|----------|----------|
| 確認済にする | 実績を確認済みにする |

## 页面交互

### 导航路径

- 来自 `p0-共通ナビゲーション` 的「実績確認」菜单点击
- 来自 `p2-発注検索`  或 `p3-発注詳細`  的相关新闻链接

### 操作流程

1. 查看実績情報
2. 查看実績詳細
3. 点击「確認済にする」
4. 确认完成
