# p2-発注検索 - 基础信息

## 页面概述

| 属性 | 值 |
|------|-----|
| 页面ID | p2 |
| 页面名称 | 発注検索 |
| 页面类型 | 搜索页面 |
| 所属模块 | 受注管理 |
| 设备类型 | PC / SP |

## 功能概要

- AzureADでのログインに成功すると、自動的に本システムへ遷移したときに表示される画面。本画面がTOPとなる
- 発注情報の検索ができ、インフォメーション（お知らせ）が表示される

## 核心功能模块

### 1. Information（お知らせ）

| 功能 | 说明 |
|------|------|
| お知らせ表示 | ベースからのお知らせを表示する |
| リンク機能 | お知らせ文のリンクを押下すると、該当する「発注詳細」「実績確認」「請求書アップロード」画面に遷移できる |

### 2. 発注検索

#### 搜索字段

| 字段名 | 字段类型 | 必填 | 说明 |
|--------|----------|------|------|
| 年月選択 | 下拉选择 | 否 | プルダウンより対象の年月が選択できる |
| 受注ID | 文本 | 否 | 対象の受注IDを入力する |
| 依頼ベース | 下拉选择 | 否 | プルダウンより対象のベースが選択できる |
| 状態 | 复选框 | 否 | チェックボックスで選択できる |

### 3. 受注内容確認（検索結果）

#### 列表字段

| 字段名 | 说明 |
|--------|------|
| 年月 | 対象発注の年月を表示する |
| 依頼ベース | 対象発注の依頼ベース名を表示する |
| 受注ID | 対象発注の発注IDを表示する |
| ステータス | 対象発注の状況をステータスで表示する |
| version | 現在の指示書のversion（数字）が表示される |
| 指示書受領 | ベースから指示書を受領した日時が表示される |
| 承諾 | 対象発注をパートナーが承諾した日時が表示される |

## 页面交互

### 导航路径

- 来自 `p0-共通ナビゲーション` 的「受注管理」菜单点击
- AzureAD登录成功后自动跳转到此页面（TOP页面）

### 页面跳转

- 点击お知らせ链接 → `p3-発注詳細` / `p4-実績確認` / `p5-請求書アップロード`
- 点击検索按钮 → 显示符合条件的订单列表

## 操作按钮

| 按钮名称 | 功能说明 |
|----------|----------|
| 検索 | 検索条件にマッチした受注情報を一覧で表示する |
