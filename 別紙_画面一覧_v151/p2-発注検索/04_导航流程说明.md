# p2-発注検索 - 导航流程说明

## 页面在系统中的位置

```
輸送パートナーシステム
├── 共通ナビゲーション (p0)
│   ├── 受注管理 → p2-発注検索 (当前页面)
│   ├── 実績確認 → p4-実績確認
│   ├── 請求確認 → p5-請求検索
│   └── ログアウト → p1-スタート
```

## 进入当前页面的导航路径

### 路径1：从主菜单进入（TOP页面）

```
パートナー用户登录系统
  ↓
AzureAD认证成功
  ↓
自动跳转到 p2-発注検索 (本画面がTOPとなる)
  ↓
显示订单搜索页面
```

### 路径2：从主菜单进入

```
用户在任意页面
  ↓
点击左侧菜单「受注管理」
  ↓
进入 p2-発注検索 页面
```

## 从当前页面出去的导航路径

### 跳转到其他页面

| 目标页面 | 触发条件 | 跳转方式 |
|----------|----------|----------|
| p3-発注詳細 | 点击お知らせ链接或订单链接 | 直接跳转 |
| p4-実績確認 | 点击相关新闻链接 | 直接跳转 |
| p5-請求書アップロード | 点击相关新闻链接 | 直接跳转 |
| p0-共通ナビゲーション | 点击左侧菜单其他项 | 直接跳转 |

## 面包屑导航

```
ホーム > 受注管理 > 発注検索
```

## 页面状态管理

### URL参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| year_month | string | 预设年月 |
| order_id | string | 预设订单ID |
| base | string | 预设ベース |
| status | string | 预设状态 |

### 页面状态

| 状态类型 | 状态值 | 说明 |
|----------|--------|------|
| view | 初始状态 | 显示搜索表单 |
| search | 搜索中 | 执行搜索 |
| result | 结果状态 | 显示搜索结果 |
| empty | 空结果 | 无匹配结果 |
