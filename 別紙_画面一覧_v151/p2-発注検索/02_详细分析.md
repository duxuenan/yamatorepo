# p2-発注検索 - 详细分析

## 页面结构层次

```
輸送パートナーシステム
├── 受注管理
│   ├── Information（お知らせ）
│   │   ├── お知らせ本文
│   │   └── リンク（発注詳細/実績確認/請求書アップロード）
│   ├── 発注検索
│   │   ├── 年月選択
│   │   ├── 受注ID
│   │   ├── 依頼ベース
│   │   ├── 状態
│   │   └── 検索ボタン
│   └── 受注内容確認
│       ├── 年月
│       ├── 依頼ベース
│       ├── 受注ID
│       ├── ステータス
│       ├── version
│       ├── 指示書受領
│       └── 承諾
```

## 字段详细说明

### お知らせ区域

| 字段标识 | 字段名称 | 数据类型 | 业务规则 |
|----------|----------|----------|----------|
| notice_id | お知らせID | string | お知らせの識別子 |
| notice_title | タイトル | string | お知らせのタイトル |
| notice_body | 本文 | string | お知らせの内容 |
| related_page | 関連ページ | enum | 発注詳細/実績確認/請求書アップロード |

### 搜索区域

| 字段标识 | 字段名称 | 数据类型 | 验证规则 | 业务规则 |
|----------|----------|----------|----------|----------|
| year_month | 年月選択 | select | 可选 | プルダウンより対象の年月が選択できる |
| order_id | 受注ID | string | 可选，格式验证 | 対象の受注IDを入力する |
| base | 依頼ベース | select | 可选 | プルダウンより対象のベースが選択できる |
| status | 状態 | checkbox[] | 可选 | チェックボックスで選択できる |

### 结果列表区域

| 字段标识 | 字段名称 | 数据类型 | 来源 | 业务规则 |
|----------|----------|----------|------|----------|
| order_date | 年月 | string | システム | 対象発注の年月を表示する |
| base_name | 依頼ベース | string | マスタ | 対象発注の依頼ベース名を表示する |
| order_id | 受注ID | string | システム | 対象発注の発注IDを表示する |
| status | ステータス | enum | システム | 対象発注の状況をステータスで表示する |
| version | version | number | システム | 現在の指示書のversion（数字）が表示される |
| instruction_received | 指示書受領 | datetime | システム | ベースから指示書を受領した日時が表示される |
| commitment | 承諾 | datetime | システム | 対象発注をパートナーが承諾した日時が表示される |

## 交互流程详细说明

### 1. 页面加载流程

```
页面加载完成
  ↓
显示Information区域（お知らせ一覧）
  ↓
显示搜索表单
  ↓
如果有预设搜索条件，自动执行搜索
  ↓
显示搜索结果列表
```

### 2. 搜索操作流程

```
输入搜索条件（可输入多个条件）
  ↓
点击「検索」按钮
  ↓
执行搜索查询
  ↓
显示符合条件的订单列表
```

### 3. お知らせ点击流程

```
用户看到お知らせ列表
  ↓
点击感兴趣のお知らせ链接
  ↓
跳转到对应的详情页面
```

## 弹窗/特殊组件说明

| 属性 | 值 |
|------|-----|
| 触发方式 | 点击お知らせ标题 |
