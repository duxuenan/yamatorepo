# p2-発注検索 - README

## 页面概述

**页面ID**: p2
**页面名称**: 発注検索
**页面类型**: 搜索页面
**所属模块**: 受注管理
**设备类型**: PC / SP

## 功能简介

- AzureADでのログインに成功すると、自動的に本システムへ遷移したときに表示される画面。本画面がTOPとなる
- 発注情報の検索ができ、インフォメーション（お知らせ）が表示される

## 核心功能

### 1. Information（お知らせ）
- ベースからのお知らせを表示する
- リンクを押下すると、該当する「発注詳細」「実績確認」「請求書アップロード」画面に遷移できる

### 2. 発注検索
- 年月選択: プルダウンより対象の年月が選択できる
- 受注ID: 対象の受注IDを入力する
- 依頼ベース: プルダウンより対象のベースが選択できる
- 状態: チェックボックスで選択できる

### 3. 受注内容確認
- 年月、依頼ベース、受注ID、ステータス、version、指示書受領、承諾

## 操作流程

1. 查看頁面上的Information（お知らせ）
2. 输入搜索条件
3. 点击「検索」按钮
4. 查看搜索结果
5. 点击订单跳转到详情页

## 特殊业务规则

- 登录成功后自动跳转到此页面（TOP页面）
- お知らせ链接直接跳转到对应的详情页面
- 支持多条件组合搜索
