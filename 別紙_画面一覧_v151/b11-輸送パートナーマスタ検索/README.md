# b11-輸送パートナーマスタ検索 画面分析文档

## 文档概述

本文档是对 Excel 文件 `別紙_画面一覧_v151.xlsx` 中 `b11-輸送パートナーマスタ検索` 工作表的详细分析。

## 画面基本信息

| 项目 | 内容 |
|------|------|
| **工作表名称** | b11-輸送パートナーマスタ検索 |
| **界面编号** | b11 |
| **画面名称** | 輸送パートナーマスタ検索 |
| **画面类型** | PC |
| **功能概述** | 输送合作伙伴主数据搜索页面 |

## 功能描述

b11-輸送パートナーマスタ検索 是合作伙伴主数据搜索页面：

1. **アカウント検索** - 輸送パートナーのアカウント情報を検索
2. **検索結果表示** - 検索結果を一覧で表示
3. **詳細跳转** - 会社情報の詳細画面に遷移

## 核心功能

| 序号 | 功能名称 | 功能说明 |
|------|----------|----------|
| 1 | アカウント検索 | 多条件搜索（会社/名前/電話/利用） |
| 2 | 検索結果 | 4字段列表展示 |
| 3 | 详情跳转 | 点击会社名→b12-詳細 |

## 搜索条件

| 条件 | 类型 | 说明 |
|------|------|------|
| 会社コード | 入力 | 本システムで設定した会社コード |
| 会社名 | 入力 | 本システムで設定した会社名 |
| 電話番号 | 入力 | 本システムで設定した電話番号 |
| システム利用 | ラジオボタン | あり/なし |

## 列表字段

| 字段 | 说明 | 交互 |
|------|------|------|
| 会社コード | 本システムで設定した会社コード | - |
