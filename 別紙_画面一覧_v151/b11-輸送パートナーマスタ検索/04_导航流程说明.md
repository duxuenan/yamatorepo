# b11-輸送パートナーマスタ検索 - 导航流程说明

## 1. 页面导航概述

### 1.1 页面定位
b11-輸送パートナーマスタ検索 是系统的**合作伙伴搜索页面**。

### 1.2 导航类型
| 导航类型 | 说明 |
|----------|------|
| **功能页面** | 提供搜索和列表功能 |
| **列表页面** | パートナー信息入口 |

## 2. 入口分析

### 2.1 直接入口
| 入口方式 | 触发条件 | 目标画面 |
|----------|----------|----------|
| 菜单点击 | b10-マスタ管理 | b11-輸送パートナーマスタ検索 |
| URL直接访问 | 携带查询参数 | b11-輸送パートナーマスタ検索 |

## 3. 出口分析

### 3.1 页面跳转出口
| 当前画面 | 触发条件 | 目标画面 | 说明 |
|----------|----------|----------|------|
| b11-輸送パートナーマスタ検索 | 点击会社名 | b12-輸送パートナー詳細（会社） | 详情页面 |

### 3.2 出口决策流程
```
[点击会社名]
        │
        ▼
[获取パートナーID]
        │
        ▼
[跳转到b12-詳細]
```

```

        │
        ▼

        │
        ▼
[点击「検索」ボタン]
        │
        ▼
[查询数据库]
        │
        ▼
[显示検索結果]
```

## 5. 导航状态管理

### 5.1 页面状态
| 状态 | 条件 | 导航行为 |
|------|------|----------|
| **初始状态** | 首次访问 | 显示空搜索区域 |
| **搜索状态** | 执行搜索 | 显示搜索结果 |
| **无结果** | 无匹配数据 | 显示空提示 |

### 5.2 状态转换
```
[页面加载]
        │
        ▼
[等待用户输入]
        │
        ▼
[执行搜索]
        │
        ▼
[显示结果/空结果]
```
