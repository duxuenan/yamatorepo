# b11-輸送パートナーマスタ検索 - 详细分析

## 1. 页面结构分析

### 1.1 页面构成
b11-輸送パートナーマスタ検索 页面由以下区域构成：

| 区域 | 内容 | 说明 |
|------|------|------|
| 検索条件区域 | ⑴アカウント検索 | 搜索条件输入 |
| 検索結果区域 | ⑵検索結果 | 列表展示 |

### 1.2 元素详细说明

#### ⑴アカウント検索
| 项目 | 内容 |
|------|------|
| **元素类型** | 搜索区域 |
| **包含条件** | 会社コード/会社名/電話番号/システム利用 |

#### ⑵検索結果
| 项目 | 内容 |
|------|------|
| **元素类型** | 列表区域 |
| **显示内容** | パートナー相关信息 |

## 2. 搜索条件分析

### 2.1 条件清单
| 条件名称 | 输入类型 | 说明 |
|----------|----------|------|
| 会社コード | 入力 | 本システムで設定した会社コード |
| 会社名 | 入力 | 本システムで設定した会社名 |
| 電話番号 | 入力 | 本システムで設定した電話番号 |
| システム利用 | ラジオボタン | 〇あり / 〇なし |

### 2.2 搜索规则
| 规则 | 说明 |
|------|------|
| システム利用未選択 | 検索可能（默认值） |

## 3. 列表字段分析

### 3.1 显示字段
| 字段名 | 说明 | 交互 |
|--------|------|------|
| 会社コード | 本システムで設定した会社コード | - |
| 会社名 | 本システムで設定した会社名 | クリック→b12-詳細 |
| 電話番号 | 本システムで設定した電話番号 | - |
| システム利用 | 本システムの利用あり/なし | - |

### 3.2 显示规则
| 字段 | 显示规则 |
|------|----------|
| システム利用 | あり / なし |
