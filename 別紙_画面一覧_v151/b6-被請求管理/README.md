# b6-被請求管理 画面分析文档

## 文档概述

本文档是对 Excel 文件 `別紙_画面一覧_v151.xlsx` 中 `b6-被請求管理` 工作表的详细分析。

## 画面基本信息

| 项目 | 内容 |
|------|------|
| **工作表名称** | b6-被請求管理 |
| **界面编号** | b6 |
| **画面名称** | 被請求管理 |
| **画面类型** | PC |
| **功能概述** | 被请求（账单）信息管理和下载 |

## 功能描述

b6-被請求管理 是账单管理页面：

1. **被請求一覧** - 被請求情報を一覧で表示
2. **詳細跳转** - 本画面から詳細画面に遷移できる
3. **請求書ダウンロード** - 一括または個別ダウンロード
4. **ステータス管理** - チェックのみ/一括ステータス変更

## 核心功能

| 序号 | 功能名称 | 功能说明 |
|------|----------|----------|
| 1 | 絞り込み | 多条件筛选（年月/コード/名前/ステータス） |
| 2 | 被請求一覧 | 12字段账单列表展示 |
| 3 | プレビュー | 预览账单文件 |
| 4 | ダウンロード | チェックのみ/一括ダウンロード |
| 5 | ステータス変更 | チェックのみ/一括ステータス変更 |
| 6 | 詳細跳转 | b7-被請求詳細 |

## 筛选条件

| 条件 | 类型 | 说明 |
|------|------|------|
| 被請求年月 | プルダウン | 选择対象年月 |
| 会社コード | 入力 | 会社コード |
| 会社名 | 入力 | 会社名 |
| ステータス | プルダウン | 未提出/要確認/チェック未/チェック済/請求完了 |

## 状态说明

| 状态 | 说明 |
|------|------|
| 未提出 | パートナー未上传账单 |
| 要確認 | 金额与実績不匹配 |
| チェック未 | 未确认 |
| チェック済 | 已确认 |
| 請求完了 | 已上传到外部系统 |
