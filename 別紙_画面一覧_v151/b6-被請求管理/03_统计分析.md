## 3. 状态流程分析

### 3.1 状态流转
```
未提出 → 要確認 → チェック未 → チェック済 → 請求完了
   ↓                                    ↑
   └─────────── 代理上传 ───────────────┘
```

### 3.2 状态触发条件
| 状态 | 触发条件 |
|------|----------|
| 未提出 | 无账单上传 |
| 要確認 | 金额不匹配 |
| チェック未 | 未确认 |
| チェック済 | 已确认 |
| 請求完了 | 已上传到外部系统 |
