# b6-被請求管理 - 详细分析

## 1. 页面结构分析

### 1.1 页面构成
b6-被請求管理 页面由以下区域构成：

| 区域 | 内容 | 说明 |
|------|------|------|
| 筛选区域 | ⑴絞り込み | 多条件筛选 |
| 被請求一覧区域 | ⑵被請求一覧 | 账单列表展示 |
| 下载功能区域 | ⑶⑷ダウンロード | 账单下载 |
| 状态变更区域 | ⑸⑹ステータス変更 | 状态管理 |

### 1.2 元素详细说明

#### ⑴絞り込み
| 项目 | 内容 |
|------|------|
| **元素类型** | 筛选区域 |
| **包含条件** | 被請求年月/会社コード/会社名/ステータス |

#### ⑵被請求一覧
| 项目 | 内容 |
|------|------|
| **元素类型** | 列表区域 |
| **显示内容** | 账单相关字段 |

## 2. 筛选条件分析

### 2.1 条件清单
| 条件名称 | 输入类型 | 说明 |
|----------|----------|------|
| 被請求年月 | プルダウン | 选择対象年月 |
| 会社コード | 入力 | 輸送パートナーの会社コード |
| 会社名 | 入力 | 輸送パートナーの名称 |
| ステータス | プルダウン | 选择ステータス |

### 2.2 ステータス选项
| 选项 | 说明 |
|------|------|
| 未提出 | 尚未上传账单 |
| 要確認 | 金额不一致需要确认 |
| チェック未 | 未检查 |
| チェック済 | 已检查 |
| 請求完了 | 请求完成 |

## 3. 列表字段分析

### 3.1 显示字段
| 字段名 | 说明 | 交互 |
|--------|------|------|
| チェックボックス | 选择 | 批量操作 |
| 会社コード | 輸送パートナーのコード | クリック→b7-被請求詳細 |
| 会社名 | 輸送パートナーの名称 | - |
| ステータス | 請求状態 | - |
| 金額チェック | OK/NG/- | - |
| 実績合計金額 | システム保持の合計金額 | - |
| 請求書合計金額 | OCR読み取り金額 | - |
| 金額差分 | 差分金額 | - |
| ファイル名 | 請求書ファイル名 | プレビュー |
| アクション履歴 | アクション履歴 | 表示 |
| 更新日時 | 更新日時 | - |
| 更新者 | 更新ユーザー | - |

### 3.2 状态说明
| 状态 | 条件 |
|------|------|
| 未提出 | パートナー未上传、ベース也未代理上传 |
| 要確認 | 金额与実績合计不一致 |
| チェック未 | 未点击「確認OK」按钮 |
| チェック済 | 已点击「確認OK」按钮 |
| 請求完了 | 已上传到ビズインテグラル |

## 4. 功能操作分析

### 4.1 下载功能
| 功能 | 操作 | 说明 |
|------|------|------|
| チェックのみダウンロード | 选中+点击按钮 | 仅下载选中账单 |
| 一括ダウンロード | 点击按钮 | 仅下载チェック済账单 |

### 4.2 状态变更功能
| 功能 | 操作 | 说明 |
|------|------|------|
| チェックのみステータス変更 | 选中+点击按钮 | 选中→請求完了 |
| 一括ステータス変更 | 点击按钮 | チェック済→請求完了 |

### 4.3 预览功能
| 功能 | 操作 | 说明 |
|------|------|------|
| プレビュー | 点击链接 | 预览账单文件 |
