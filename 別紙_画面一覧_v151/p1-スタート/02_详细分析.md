# p1-スタート - 详细分析

## 页面结构层次

```
輸送パートナーシステム
├── スタート画面 (p1)
│   ├── ログインフォーム
│   │   ├── ユーザーID入力
│   │   ├── パスワード入力
│   │   └── ログインボタン
│   └── パスワード再設定リンク
```

## 字段详细说明

### 登录表单

| 字段标识 | 字段名称 | 数据类型 | 验证规则 | 业务规则 |
|----------|----------|----------|----------|----------|
| user_id | ユーザーID | string | 必填，格式验证 | 登録されているユーザーIDを入力 |
| password | パスワード | string | 必填，最小长度 | 大文字小文字数字を含む8文字以上 |

### 按钮功能

| 按钮名称 | 功能 | 后续操作 |
|----------|------|----------|
| ログイン | 認証実行 | 成功→p0、失敗→エラー表示 |
| パスワードを忘れた方 | パスワード再設定 | p1-パスワード再設定へ |

## 交互流程详细说明

### 1. 登录流程

```
页面加载完成
  ↓
显示登录表单
  ↓
用户输入凭据
  ↓
点击「ログイン」
  ↓
验证凭据
  ↓
认证成功 → 跳转到 p0
  ↓
认证失败 → 显示错误消息
```

### 2. 登出后流程

```
用户在任意页面点击「ログアウト」
  ↓
清除会话
  ↓
跳转到 p1-スタート
  ↓
显示登录界面
  ↓
用户重新登录
```

## 错误处理

| 错误类型 | 错误消息 | 处理方式 |
|----------|----------|----------|
| 用户名错误 | ユーザーIDが正しくありません | 提示用户检查输入 |
| 密码错误 | パスワードが正しくありません | 提示用户重试 |
| 账户锁定 | アカウントがロックされています | 提示联系管理员 |
| 会话过期 | セッションが切れました | 重新登录 |
