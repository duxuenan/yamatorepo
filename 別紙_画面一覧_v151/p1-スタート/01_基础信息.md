# p1-スタート - 基础信息

## 页面概述

| 属性 | 值 |
|------|-----|
| 页面ID | p1 |
| 页面名称 | スタート |
| 页面类型 | 起始页面 / 登录页面 |
| 所属模块 | 共通 |
| 设备类型 | PC / SP |

## 功能概要

- パートナーシステムのスタート画面（ログイン画面）
- ユーザー認証を行う
- 認証成功后、システム利用可能な状态になる

## 核心功能模块

### 1. 登录表单

| 字段名 | 字段类型 | 必填 | 说明 |
|--------|----------|------|------|
| ユーザーID | 文本 | 是 | ユーザーIDを入力 |
| パスワード | 密码 | 是 | パスワードを入力 |

### 2. 操作按钮

| 按钮名称 | 功能说明 |
|----------|----------|
| ログイン | 認証を実行してシステムにログイン |
| パスワードを忘れた方 | パスワード再設定画面へ遷移 |

## 页面交互

### 导航路径

- 初始访问: 直接访问p1页面
- 登录成功: → p0-共通ナビゲーション
- 登出完成: → p1-スタート（ログアウト状態）

### 登录流程

1. 输入ユーザーID和パスワード
2. 点击「ログイン」按钮
3. 验证凭据
4. 成功 → p0-共通ナビゲーション
5. 失败 → 显示错误消息
