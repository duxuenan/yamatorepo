# p1-スタート - README

## 页面概述

**页面ID**: p1
**页面名称**: スタート
**页面类型**: 起始页面 / 登录页面
**所属模块**: 共通
**设备类型**: PC / SP

## 功能简介

- パートナーシステムのスタート画面（ログイン画面）
- ユーザー認証を行う
- 認証成功后、システム利用可能な状态になる

## 核心功能

### 登录表单
- ユーザーID: ユーザーIDを入力
- パスワード: パスワードを入力
- ログインボタン: 認証を実行

### 其他功能
- パスワードを忘れた方: パスワード再設定画面へ遷移

## 操作流程

1. 输入ユーザーID和パスワード
2. 点击「ログイン」按钮
3. 认证成功 → 进入 p0-共通ナビゲーション
4. 认证失败 → 显示错误消息

## 特殊业务规则

- 会话过期后自动重定向到登录页面
- 连续登录失败5次后账户锁定30分钟
- 登出后清除会话并返回登录页面
