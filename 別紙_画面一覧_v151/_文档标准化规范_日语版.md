## 📁 ファイル命名規範

### 1.1 標準ディレクトリ構造

全ワークシートの分析は以下のファイル構造を統一的に使用する：

```
[ワークシート名]/
├── README.md                 # 文書概要とインデックス（必須）
├── 01_基礎情報.md           # 基本情報とデータ構造（必須）
├── 02_詳細分析.md           # 機能と業務ロジック分析（必須）
├── 03_統計分析.md           # 定量統計データ（必須）
├── 05_設計仕様.md           # 開発実装設計仕様（新規追加）
├── data_元データ.csv         # 元データエクスポート（必須）
└── [特殊補完文書].md         # ワークシートタイプに応じた特殊分析（任意）
        └── 04_ナビゲーションフロー説明.md
```

- 標準Markdown構文を使用

- 読みやすさ向上のためemojiを使用（📊、📋、🔍、✅、⚠️等）

```yaml

```

```typescript
// TypeScriptサンプル
```

- 見出しレベル1（#）：文書タイトル
