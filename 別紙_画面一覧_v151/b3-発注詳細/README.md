# b3-発注詳細 画面分析文档

## 文档概述

本文档是对 Excel 文件 `別紙_画面一覧_v151.xlsx` 中 `b3-発注詳細` 工作表的详细分析。

## 画面基本信息

| 项目 | 内容 |
|------|------|
| **工作表名称** | b3-発注詳細 |
| **界面编号** | b3 |
| **画面名称** | 発注詳細 |
| **画面类型** | PC |
| **功能概述** | 订单详情页面，提供指示书编辑和发送功能 |

## 功能描述

b3-発注詳細 是订单详情页面：

1. **輸送パートナー情報** - 対象輸送パートナー情報表示
2. **状態表示** - 发注対象の進捗状況表示
3. **月間運行指示書** - 参照/編集/送付功能
4. **アクション履歴** - 本画面での行動登録/表示
5. **メッセージ** - 輸送パートナーとのメッセージやり取り
