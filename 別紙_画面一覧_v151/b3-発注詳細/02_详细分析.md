# b3-発注詳細 - 详细分析

## 1. 页面结构分析

### 1.1 页面构成
b3-発注詳細 页面由以下区域构成：

| 区域 | 内容 | 说明 |
|------|------|------|
| 輸送パートナー区域 | ⑴輸送パートナー | 対象輸送パートナー情報 |
| 状態区域 | ⑵状態 | 发注対象進捗状況 |
| 月間運行指示書区域 | ⑶月間運行指示書 | 指示書编辑/送付 |
| アクション履歴区域 | ⑷状態・アクション履歴 | 行为履历 |
| メッセージ区域 | ⑸メッセージ | 消息交流 |

### 1.2 元素详细说明

#### ⑴輸送パートナー
| 项目 | 内容 |
|------|------|
| **元素类型** | 信息显示区域 |
| **显示内容** | 輸送パートナーID、輸送パートナー名 |

#### ⑵状態
| 项目 | 内容 |
|------|------|
| **元素类型** | 状态显示 |
| **显示内容** | 会社コード、ステータス、version、指示書送付、承諾 |

#### version字段说明
| 场景 | 行为 |
|------|------|
| 默认显示 | 显示月度运行指示书的最新version（数字） |
| 表格上方version选择 | プルダウン选择不同的version进行查看/编辑 |

#### ⑶月間運行指示書
| 项目 | 内容 |
|------|------|
| **元素类型** | 表格编辑区域 |
| **功能** | version選択、編集、変更保存 |
| **子功能** | 指示書送付、代理承諾 |

#### ⑷状態・アクション履歴
| 项目 | 内容 |
|------|------|
| **元素类型** | 历史记录 |
| **功能** | 本画面での行動登録/表示 |

#### ⑸メッセージ
| 项目 | 内容 |
|------|------|
| **元素类型** | 消息区域 |
| **功能** | 表示、入力、送信、ファイル添付 |

## 2. 月間運行指示書表格字段

### 2.1 表格列
| 字段 | 说明 | 交互 |
|------|------|------|
| チェックボックス | 选择指示 | 一括振り分け |
| 支点/営業所 | 分支选择 | プルダウン选择 |
| ステータス | 承認状態 | 表示/切り替え |
| 線便 | 线路编号 | 表示 |
| AM/発 | 上午发货 | 〇/空栏切换 |
| 着 | 到达 | 〇/空栏切换 |
| 系統 | 系统 | 表示 |
| 線便名 | 线路名称 | 表示 |
| 入庫場所 | 入库场所 | 表示 |
| 入庫時間 | 入库时间 | 表示 |
| 使用車両(トン) | 使用车辆 | 表示 |
| 火/水/木/金/土/日 | 星期日期 | 〇/空栏切换 |
| 指示書送付 | 輸送パートナーへ指示書を送付した日時 | 表示（只读） |
| 承諾 | 輸送パートナーが承諾した日時 | 表示（只读） |

### 2.2 指示書送付和承諾字段
| 字段 | 数据类型 | 说明 |
|------|----------|------|
| 指示書送付 | datetime | 輸送パートナーへ指示書を送付した日時を表示する |
| 承諾 | datetime | 輸送パートナーが承諾した日時を表示する（未承諾の場合は空白） |

### 2.2 状态标识
| 标识 | 说明 |
|------|------|
| 承認済 | 已承认 |
| 未承認 | 未承认 |
| 差戻し | 退回 |
| ＊ | 标记 |

## 3. 交互功能分析

### 3.1 指示书编辑功能
| 功能 | 操作 | 说明 |
|------|------|------|
| 日付切换 | 点击〇/空栏 | 显示吹き出し |
| 吹き出し入力 | 事由/変更後 | 输入变更内容 |
| 強調表示 | 背景強調 | 鼠标over显示内容 |
| 行追加 | 「＋追加」链接 | 添加空行 |
| 保存 | 「変更を保存」按钮 | 保存编辑内容 |

### 3.2 送付功能
| 功能 | 操作 | 说明 |
|------|------|------|
| 指示書送付 | 「輸送パートナーに送付する」按钮 | 发送指示书 |
| 代理承諾 | 「代理承諾」ボタン | 代替承诺 |

### 3.3 サブアカウント振り分け
| 功能 | 操作 | 说明 |
|------|------|------|
| プルダウン | 选择支店/営業所 | 单一选择 |
| チェックボックス | 批量选择 | 一括振り分け |
