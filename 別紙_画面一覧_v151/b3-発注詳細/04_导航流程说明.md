# b3-発注詳細 - 导航流程说明

## 1. 页面导航概述

### 1.1 页面定位
b3-発注詳細 是系统的**详情编辑页面**，提供订单详细信息和指示书管理功能。

### 1.2 导航类型
| 导航类型 | 说明 |
|----------|------|
| **详情页面** | 展示和编辑详细数据 |
| **功能页面** | 多功能集成（编辑/送付/消息） |

## 2. 入口分析

### 2.1 直接入口
| 入口方式 | 触发条件 | 目标页面 |
|----------|----------|----------|
| 列表跳转 | b2-発注検索点击公司代码 | b3-発注詳細 |
| URL直接访问 | 携带订单ID | b3-発注詳細 |

### 2.2 入口参数
| 参数 | 类型 | 说明 |
|------|------|------|
| orderId | string | 订单ID |
| companyCode | string | 公司代码 |

## 3. 出口分析

### 3.1 页面跳转出口
| 当前页面 | 触发条件 | 目标画面 | 说明 |
|----------|----------|----------|------|
| b3-発注詳細 | 返回按钮 | b2-発注検索 | 返回列表 |
| b3-発注詳細 | 点击消息链接 | 相关详情 | 跳转关联页面 |

### 3.2 出口决策流程
```
[详情页面操作]
        │
        ▼
[编辑完成/取消]
        │
        ▼
[返回列表]
```

## 5. 导航状态管理

### 5.1 页面状态
| 状态 | 条件 | 导航行为 |
|------|------|----------|
| **浏览模式** | 默认 | 只读显示 |
| **编辑模式** | 点击編集 | 可编辑状态 |
| **发送中** | 点击送付 | 显示loading |
| **已完成** | 操作完成 | 显示成功提示 |

### 5.2 状态转换
```
[浏览模式]
        │
        ▼
[点击編集]
        │
        ▼
[编辑模式]
        │
    ┌───┴───┐
    │       │
  保存     取消
    │       │
    ▼       ▼
[保存成功] [浏览模式]
```

## 7. 特殊导航场景

### 7.1 权限控制
| 场景 | 处理方式 |
|------|----------|
| 无权限访问 | 重定向或显示错误 |
| 只读权限 | 隐藏编辑按钮 |

### 7.2 数据状态
| 场景 | 显示内容 | 用户操作 |
|------|----------|----------|
| 未承認 | 指示书状态显示 | 可编辑 |
| 承認済 | 已承认状态 | 显示[差戻し] |
| 差戻し | 退回状态 | 可重新编辑 |
