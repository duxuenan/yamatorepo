# b15-ベースアカウントマスタ詳細 - README

## 页面概述

**页面ID**: b15
**页面名称**: ベースアカウントマスタ詳細
**所属模块**: マスタ管理
**设备类型**: PC

## 功能简介

- アカウント情報を参照し、権限やアカウント有効・無効を設定することができる

## 核心功能

### 1. ユーザー情報
- ユーザーID: AzureADから取得したユーザーIDを表示する (※編集不可)
- メールアドレス: AzureADから取得したメールアドレスを表示する (※編集不可)
- ユーザー名: AzureADから取得したユーザー名を表示する (※編集不可)
- 所属: プルダウンより、該当する「ベース」または、「本社」を選択する (※入力必須)

### 2. 権限
- ● 管理者: 全権限
- 〇 役割者: 中等権限
- 〇 配車担当者: 配車業務権限
- 〇 計上担当者: 計上業務権限

### 3. アカウント有効・無効
- 有効: 本システムを利用できる
- 無効: 本システムを利用できない

## 操作说明

1. 查看用户基本信息（只读）
2. 设置所属（从ベース一覧选择）
3. 选择用户权限
4. 切换账户有效/无效状态
5. 保存设置

## 特殊业务规则

- 用户ID、メールアドレス、ユーザー名は只读的
- 如需修改只读字段，需要在AzureAD侧进行修改
- 从b14页面新增用户时，会预填AzureAD的用户信息
