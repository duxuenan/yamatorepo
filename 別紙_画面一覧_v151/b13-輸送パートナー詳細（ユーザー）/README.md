# b13-輸送パートナー詳細（ユーザー）- README

## 页面概述

**页面ID**: b13
**页面名称**: 輸送パートナー詳細（ユーザー）
**所属模块**: マスタ管理
**设备类型**: PC

## 功能简介

- 輸送パートナー会社のユーザー情報が確認できる
- ユーザー設定ができる

## 核心功能

### 1. ユーザー情報（只读）
- ユーザーID: ネコーレWEBに登録されているユーザーIDを表示する
- メールアドレス: ネコーレWEBに登録されているメールアドレスを表示する
- 個人SMS: ネコーレWEBに登録されている電話番号を表示する

### 2. ユーザー設定
- 通知送付先: ● メール / 〇 SMS
- アカウント区分: ● メイン / 〇 サブ
- サブアカウントへの連絡受領: ● する / 〇 しない

### 3. サブアカウント設定（仅サブアカウント显示）
- 支店・営業所名: 任意の支店・営業所名を入力する

### 4. アカウント有効・無効
- 有効: 本システムを利用できる
- 無効: 本システムを利用できない

## 特殊业务规则

- 从b12页面新增用户时，会预填ネコーレWEB的用户信息
- サブアカウントへの連絡受領仅在アカウント区分=メイン时显示
- 支店・営業所名仅在アカウント区分=サブ时显示且为必填
