## 进入当前页面的导航路径

### 路径1：从公司详情页面进入（点击用户ID）

```
用户访问 b12 页面
  ↓
点击利用者列表中的用户ID链接
  ↓
进入 b13-輸送パートナー詳細（ユーザー） 页面
```

### 路径2：从公司详情页面进入（新增用户）

```
用户访问 b12 页面
  ↓
点击「＋新規追加」按钮
  ↓
进行用户搜索
  ↓
点击「決定」按钮
  ↓
进入 b13-輸送パートナー詳細（ユーザー） 页面（新增模式）
```

## 页面状态管理

### URL参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| user_id | string | 用户ID |
| company_id | string | 所属公司ID |
| mode | string | edit / add |

### 页面状态

| 状态类型 | 状态值 | 说明 |
|----------|--------|------|
| mode | view | 浏览模式 |
| mode | edit | 编辑模式 |
| mode | add | 新增模式 |
