# b13-輸送パートナー詳細（ユーザー）- 基础信息

## 页面概述

| 属性 | 值 |
|------|-----|
| 页面ID | b13 |
| 页面名称 | 輸送パートナー詳細（ユーザー） |
| 页面类型 | 详情页面 |
| 所属模块 | マスタ管理 |
| 设备类型 | PC |

## 功能概要

- 輸送パートナー会社のユーザー情報が確認できる
- ユーザー設定ができる

## 核心功能模块

### 1. 輸送パートナー：ユーザー情報

#### 只读字段

| 字段名 | 字段类型 | 必填 | 说明 |
|--------|----------|------|------|
| ユーザーID | 文本 | 是 | ネコーレWEBに登録されているユーザーIDを表示する ※編集不可 |
| メールアドレス | 文本 | 是 | ネコーレWEBに登録されているメールアドレスを表示する ※編集不可 |
| 個人SMS | 文本 | 是 | ネコーレWEBに登録されている電話番号を表示する ※編集不可 |

### 2. ユーザー設定

#### 选择项

| 字段名 | 字段类型 | 选项 | 说明 |
|--------|----------|------|------|
| 通知送付先 | radio | ● メール / 〇 SMS | メールまたは、SMSを選択する ※デフォルトはメール |
| アカウント区分 | radio | ● メイン / 〇 サブ | メインのアカウントなのか、サブアカウントなのかを選択する |

#### 开关选项

| 字段名 | 字段类型 | 选项 | 说明 |
|--------|----------|------|------|
| サブアカウントへの連絡受領 | radio | ● する / 〇 しない | メインアカウントの場合、サブアカウントの連絡も受領するか、しないかを選択する |

### 3. サブアカウント設定

#### 输入字段（仅サブアカウント显示）

| 字段名 | 字段类型 | 必填 | 说明 |
|--------|----------|------|------|
| 支店・営業所名 | 文本 | 是 | 任意の支店・営業所名を入力する |

### 4. アカウント有効・無効

#### 开关

| 状态 | 说明 |
|------|------|
| 有効 | 本システムを利用できる |
| 無効 | 本システムを利用できない |

## 页面交互

### 导航路径

- 来自 `b12-輸送パートナーマスタ詳細（会社）` 页面的用户ID链接点击
- 来自 `b12-輸送パートナーマスタ詳細（会社）` 页面的＋新規追加操作

### 操作按钮

| 按钮名称 | 功能说明 |
|----------|----------|
| 保存 | 本画面で設定した内容を保存できる |

## 特殊业务规则

- 从「b12-輸送パートナー詳細（会社）」画面的「新規登録」到本画面时，用户ID和メールアドレス会预填ネコーレWEB的信息
- サブアカウントへの連絡受領：この項目はメインアカウントの場合に選択することができます
- 支店・営業所名：この項目はサブアカウントの場合に設定することができます
