# b8-CSVアップロード 画面分析文档

## 文档概述

本文档是对 Excel 文件 `別紙_画面一覧_v151.xlsx` 中 `b8-CSVアップロード` 工作表的详细分析。

## 画面基本信息

| 项目 | 内容 |
|------|------|
| **工作表名称** | b8-CSVアップロード |
| **界面编号** | b8 |
| **画面名称** | ファイルアップロード |
| **画面类型** | PC |
| **功能概述** | BOSS系统CSV文件上传页面 |

## 功能描述

b8-CSVアップロード 是CSV上传页面：

1. **CSVアップロード** - BOSSから出力したCSVファイルをアップロード
2. **ファイルチェック** - アップロードしたCSVをチェック
3. **データチェック** - データの整合性をチェック
4. **パートナーに反映** - アップロードしたデータを取り込む
5. **一斉メール送信** - 輸送パートナーに通知メールを送付
