# b8-CSVアップロード - 详细分析

## 1. 页面结构分析

### 1.1 上传方式
b8-CSVアップロード 支持两种上传方式：

| 方式 | 说明 | 対象 |
|------|------|------|
| **全件单位** | 一括で全パートナーに上传 | あらかじめベースごとに定められた単位で全件上传 |
| **個別单位** | 個別にパートナーに上传 | 個別パートナー単位で上传 |

**注意**: 上传方式由ベースごとの設定决定，システムが自動的に判断して適用する。

### 1.2 Tabs结构
b8-CSVアップロード 页面包含3个Tab：

| Tab | 上传文件 | 时机 |
|-----|----------|------|
| 発注（前月25日～26日） | 月間運行計画表 | 前月25日～26日 |
| 実績確認（当月中） | 日別傭車費一覧 | 当月中 |
| 発注＆実績確認（〆後） | 両方 | 〆後 |

### 1.3 上传方法在画面上的体现
| 场景 | 画面行为 |
|------|----------|
| 全件单位 | 選択した年月の全パートナーに対して一括上传 |
| 個別单位 | 個別パートナーごとにアップロードするファイルを選択 |

### 1.2 通用元素

#### ⑴年月選択
| 项目 | 内容 |
|------|------|
| **元素类型** | プルダウン |
| **功能描述** | 対象の年月を選択 |

#### ⑵アップロードする書類
| 项目 | 内容 |
|------|------|
| **元素类型** | 表示項目 |
| **功能描述** | アップロードする書類を表す |

#### ⑶アップロード済ファイル名
| 项目 | 内容 |
|------|------|
| **元素类型** | 表示項目 |
| **功能描述** | アップロードしたファイル名と日時を表示 |

### 1.3 上传功能

#### ⑷ファイルアップロード／再アップロード
| 项目 | 内容 |
|------|------|
| **元素类型** | ボタン |
| **功能描述** | ファイルを選択する |

#### ⑸ファイルチェック結果
| 检查项目 | 说明 |
|----------|------|
| 必須項目ブランク | 必須項目にブランクが含まれているか |
| 項目数 | 項目数があっているか |
| 文字コード | 文字コードがあっているか |
| NG処理 | ファイルチェックNGの場合は、データは取り込めない |

#### ⑹データチェック結果
| 功能 | 说明 |
|------|------|
| 差分チェック | システムにあるがCSVにないデータを表示 |
| 残すデータ選択 | チェックボックスで残すことができる |
| キャンセル | ファイルを取り込まないこともできる |

### 14 反映功能

#### ⑺パートナーに反映
| 功能 | 说明 |
|------|------|
| データ取り込み | アップロードしたデータを取り込む |
| ver.up | ver.nのカウントアップ |
| 反映日時 | 反映した日時を表示 |
| 活性化条件 | ファイルアップロードするまで非活性 |

### 1.5 邮件功能

#### ⑻一斉メール送信
| 功能 | 说明 |
|------|------|
| 通知メール | サブアカウントがない輸送パートナーに送付 |
| 対象数表示 | 対象となる輸送パートナー数を表示 |
| 非対象表示 | サブアカウント保有パートナー数は送付されない |
| 活性化条件 | ファイルアップロードするまで非活性 |
