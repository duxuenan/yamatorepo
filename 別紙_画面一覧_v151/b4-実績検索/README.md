# b4-実績検索 画面分析文档

## 文档概述

本文档是对 Excel 文件 `別紙_画面一覧_v151.xlsx` 中 `b4-実績検索` 工作表的详细分析。

## 画面基本信息

| 项目 | 内容 |
|------|------|
| **工作表名称** | b4-実績検索 |
| **界面编号** | b4 |
| **画面名称** | 実績検索 |
| **画面类型** | PC |
| **功能概述** | 业绩信息搜索和列表展示 |

## 功能描述

b4-実績検索 是业绩搜索页面：

1. **業績検索** - 業績情報の検索＆一覧表示
2. **業績詳細** - 業績情報の詳細画面に遷移できる

## 核心功能

| 序号 | 功能名称 | 功能说明 |
|------|----------|----------|
| 1 | 業績検索 | 多条件搜索（年月/ID/作業区分/状態） |
| 2 | 業績列表 | 11字段详细信息展示 |
| 3 | 详情跳转 | 点击公司コード→b5-実績詳細 |

## 搜索条件

| 条件 | 类型 | 说明 |
|------|------|------|
| 年月選択 | プルダウン | 选择対象年月 |
| 輸送パートナーID | 入力 | 輸送パートナーID |
| 実績ID | 入力 | 対象の実績ID |
| 作業区分 | チェックボックス | 運行/回送/横持/センター/物流/合計 |
| 状態 | チェックボックス | 全承諾/未承諾 |

## 列表字段

| 字段 | 说明 | 交互 |
|------|------|------|
| 会社コード | 輸送パートナーのコード | クリック→b5-実績詳細 |
| 会社名 | 輸送パートナーの名称 | - |
| サブアカウント | 保有サブアカウント名 | - |
| 作業区分 | 作業区分 | - |
| メッセージ | 未読状態 | - |
| ステータス | 承諾状態 | - |
| 最終データ更新日時 | 更新日時 | - |
| version | 最新version | - |
| 一斉メール送信 | 送信日付 | - |
| 月間運行指示書 | 承諾状況 | - |
| 日別傭車費一覧 | 承諾状況 | - |

## 导航关系
