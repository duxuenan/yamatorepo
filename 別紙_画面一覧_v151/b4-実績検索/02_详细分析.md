# b4-実績検索 - 详细分析

## 1. 页面结构分析

### 1.1 页面构成
b4-実績検索 页面由以下区域构成：

| 区域 | 内容 | 说明 |
|------|------|------|
| 検索条件区域 | ⑴業績検索 | 多条件搜索 |
| 検索結果区域 | ⑵検索結果 | 列表展示 |

### 1.2 元素详细说明

#### ⑴業績検索
| 项目 | 内容 |
|------|------|
| **元素类型** | 搜索区域 |
| **包含条件** | 年月選択/輸送パートナーID/実績ID/作業区分/状態 |

#### ⑵検索結果
| 项目 | 内容 |
|------|------|
| **元素类型** | 列表区域 |
| **显示内容** | 业绩相关字段 |

## 2. 搜索条件分析

### 2.1 条件清单
| 条件名称 | 输入类型 | 说明 |
|----------|----------|------|
| 年月選択 | プルダウン | 选择対象年月 |
| 輸送パートナーID | 入力 | 輸送パートナーID |
| 実績ID | 入力 | 対象の実績ID |
| 作業区分 | チェックボックス | 多选：運行/回送/横持/センター/物流/合計 |
| 状態 | チェックボックス | 全承諾/未承諾 |

### 2.2 作业区分选项
| 选项 | 说明 |
|------|------|
| 運行/回送 | 运行相关 |
| 横持 | 横持运输 |
| センター | 中心作业 |
| 物流 | 物流作业 |
| 合計 | 全部合计 |

## 3. 列表字段分析

### 3.1 显示字段
| 字段名 | 说明 | 交互 |
|--------|------|------|
| 会社コード | 輸送パートナーのコード | クリック→b5-実績詳細 |
| 会社名 | 輸送パートナーの名称 | - |
| サブアカウント | 保有サブアカウント名 | - |
| 作業区分 | 作業区分 | - |
| メッセージ | 未読状態表示 | - |
| ステータス | 承諾状態 | - |
| 最終データ更新日時 | 更新日時 | - |
| version | 最新version | - |
| 一斉メール送信 | 送信日付 | - |
| 月間運行指示書 | 承諾状況 | - |
| 日別傭車費一覧 | 承諾状況 | - |

### 3.2 显示规则
| 字段 | 显示规则 |
|------|----------|
| メッセージ | 未読あり表示、未読がない場合は「-」 |
| 一斉メール送信 | 未送信は「-」表示 |
| 作業区分 | 根据实际情况显示对应标签 |
