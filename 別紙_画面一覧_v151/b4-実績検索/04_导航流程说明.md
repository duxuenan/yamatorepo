# b4-実績検索 - 导航流程说明

## 1. 页面导航概述

### 1.1 页面定位
b4-実績検索 是系统的**业绩搜索页面**，提供业绩信息的检索和列表展示功能。

### 1.2 导航类型
| 导航类型 | 说明 |
|----------|------|
| **功能页面** | 提供搜索和列表功能 |
| **列表页面** | 业绩信息入口 |

## 2. 入口分析

### 2.1 直接入口
| 入口方式 | 触发条件 | 目标页面 |
|----------|----------|----------|
| 菜单点击 | b0-共通ナビゲーション | b4-実績検索 |
| URL直接访问 | 携带查询参数 | b4-実績検索 |

## 3. 出口分析

### 3.1 页面跳转出口
| 当前页面 | 触发条件 | 目标画面 | 说明 |
|----------|----------|----------|------|
| b4-実績検索 | 点击公司代码 | b5-実績詳細 | 业绩详情 |

### 3.2 出口决策流程
```
[点击会社コード]
        │
        ▼
[获取実績ID]
        │
        ▼
[跳转到b5-実績詳細]
```

## 5. 导航状态管理

### 5.1 页面状态
| 状态 | 条件 | 导航行为 |
|------|------|----------|
| **初始状态** | 首次访问 | 显示空搜索区域 |
| **搜索状态** | 执行搜索 | 显示搜索结果 |
| **无结果** | 无匹配数据 | 显示空提示 |

### 5.2 状态转换
```
[页面加载]
        │
        ▼
[等待用户输入]
        │
        ▼
[执行搜索]
        │
        ▼
[显示结果/空结果]
```
