# p5-請求検索 - 统计分析

## 請求统计

### 1. 請求金额统计

| 统计维度 | 计算方法 | 业务含义 |
|----------|----------|----------|
| 総請求額 | SUM(amount) | 总请求金额 |
| 平均請求額 | AVG(amount) | 平均请求金额 |
| 最大請求額 | MAX(amount) | 最大请求金额 |
| 最小請求額 | MIN(amount) | 最小请求金额 |

### 2. 状态分布

| 状态 | 说明 | 占比 |
|------|------|------|
| 未請求 | 未発行 | - |
| 請求済 | 発行済み | - |
| 支払済 | 支払い完了 | - |

### 示例数据

```json
{
  "billings": [
    {
      "billing_id": "BL-2024-001",
      "amount": 50000,
      "status": "請求済",
      "billing_date": "2024年1月"
    }
  ]
}
```
