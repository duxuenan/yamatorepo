# p5-請求検索 - 基础信息

## 页面概述

| 属性 | 值 |
|------|-----|
| 页面ID | p5 |
| 页面名称 | 請求検索 |
| 页面类型 | 搜索页面 |
| 所属模块 | 請求管理 |
| 设备类型 | PC / SP |

## 功能概要

- パートナー向けの請求書情報を検索できる
- 請求詳細を確認できる

## 核心功能模块

### 1. 請求検索

| 字段名 | 字段类型 | 说明 |
|--------|----------|------|
| 年月選択 | 下拉选择 | 対象の年月を選択 |
| 請求ID | 文本 | 対象の請求IDを入力 |
| ステータス | 下拉选择 | 請求のステータス |

### 2. 検索結果

| 字段名 | 说明 |
|--------|------|
| 年月 | 請求の年月 |
| 請求ID | 請求の識別子 |
| 金額 | 請求金額 |
| ステータス | 請求の状況 |
| 請求日 | 請求発行日 |

## 页面交互

### 导航路径

- 来自 `p0-共通ナビゲーション` 的「請求確認」菜单点击

### 操作流程

1. 输入搜索条件
2. 点击「検索」
3. 查看搜索结果
4. 点击请求详情跳转到詳細页面
