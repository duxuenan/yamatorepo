# p3-発注詳細 - 基础信息

## 页面概述

| 属性 | 值 |
|------|-----|
| 页面ID | p3 |
| 页面名称 | 発注詳細 |
| 页面类型 | 详情页面 |
| 所属模块 | 受注管理 |
| 设备类型 | PC / SP |

## 功能概要

- 発注の詳細情報を表示する
- パートナーは発注を承諾または却下できる

## 核心功能模块

### 1. 発注情報

| 字段名 | 说明 |
|--------|------|
| 受注ID | 発注のID |
| 年月 | 発注の年月 |
| 依頼ベース | 依頼元のベース名 |
| ステータス | 発注の現在のステータス |

### 2. 依頼内容

| 字段名 | 说明 |
|--------|------|
| 依頼内容 | ベースからの依頼詳細 |
| 依頼日 | 依頼発行日 |
| 受付期限 | 受付の期限 |

### 3. パートナー操作

| 按钮名称 | 功能说明 |
|----------|----------|
| 承諾 | 発注を承諾する |
| 却下 | 発注を却下する |
| 備考入力 | パートナーが備考を入力 |

## 页面交互

### 导航路径

- 来自 `p2-発注検索` 的订单点击
- 来自 `p0-共通ナビゲーション` 的相关新闻链接

### 操作流程

1. 查看発注詳細
2. 入力備考（可选）
3. 点击「承諾」或「却下」
4. 确认操作
5. 返回p2页面
