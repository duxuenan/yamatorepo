# p3-発注詳細 - README

## 页面概述

**页面ID**: p3
**页面名称**: 発注詳細
**页面类型**: 详情页面
**所属模块**: 受注管理
**设备类型**: PC / SP

## 功能简介

- 発注の詳細情報を表示する
- パートナーは発注を承諾または却下できる

## 核心功能

- 発注情報表示
- 依頼内容表示
- 備考入力
- 承諾/却下操作

## 相关页面

- **上级页面**: p2-発注検索
- **下级页面**: 无

## 操作流程

1. 查看発注詳細
2. 入力備考（可选）
3. 点击「承諾」或「却下」
4. 确认操作
5. 返回p2页面
