## 1. 页面复杂度评估

### 1.1 复杂度指标
| 指标 | 评估值 | 说明 |
|------|--------|------|
| **功能复杂度** | ⭐ 低 | 仅包含登录入口和认证跳转功能 |
| **页面元素数量** | 极少 | 仅1个按钮和少量文本 |
| **交互复杂度** | 低 | 单次点击完成主要交互 |
| **数据处理量** | 无 | 仅做页面跳转 |
| **业务规则数量** | 少 | 主要为认证流程规则 |

| 画面 | 复杂度 | 相对比较 |

| b2-発注検索 | 高 | 复杂查询和列表功能 |
