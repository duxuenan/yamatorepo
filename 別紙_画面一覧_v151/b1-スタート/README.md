# b1-スタート 画面分析文档

## 文档概述

本文档是对 Excel 文件 `別紙_画面一覧_v151.xlsx` 中 `b1-スタート` 工作表的详细分析。

## 画面基本信息

| 项目 | 内容 |
|------|------|
| **工作表名称** | b1-スタート |
| **界面编号** | b1 |
| **画面名称** | スタート |
| **画面类型** | PC |
| **功能概述** | 系统入口页面，提供 AzureAD 认证功能 |

## 功能描述

b1-スタート 是系统的起始页面：

1. **系统入口** - 本システムにURLよりアクセスしたとき、はじめに表示される画面
2. **认证跳转** - 本画面からAzureAD側のログイン画面に遷移できる

## 核心功能

| 序号 | 功能名称 | 功能说明 |
|------|----------|----------|
| 1 | 登录入口 | 显示系统登录入口 |
| 2 | AzureAD认证 | 跳转到 AzureAD 进行身份验证 |

## 技术要点

- **认证方式**: Azure Active Directory (OAuth 2.0 / OIDC)
- **前端框架**: React 18.x + TypeScript + Ant Design
- **后端框架**: Spring Boot 3.x + Java 17
- **设备支持**: PC 端完整支持，SP 端不支持

## 导航关系

### 入口
- 直接访问系统 URL

### 跳转
- 点击登录按钮 → AzureAD 登录页面（外部）
- 认证成功 → 系统主页面
