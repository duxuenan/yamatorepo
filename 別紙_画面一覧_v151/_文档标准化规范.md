## 📁 文件命名规范

### 1.1 标准目录结构

所有工作表分析统一使用以下文件结构：

```
[工作表名称]/
├── README.md                 # 文档总览和索引（必选）
├── 01_基础信息.md           # 基本信息和数据结构（必选）
├── 02_详细分析.md           # 功能和业务逻辑分析（必选）
├── 03_统计分析.md           # 量化统计数据（必选）
├── 05_设计规范.md           # 开发实现设计规范（新增）
├── data_原始数据.csv         # 原始数据导出（必选）
└── [特殊补充文档].md         # 根据工作表类型的特殊分析（可选）
        └── 04_导航流程说明.md
```
